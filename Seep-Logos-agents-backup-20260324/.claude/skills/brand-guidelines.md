# ブランドガイドライン（Brand Guidelines）

## このスキルの目的
Seep Logos のブランドアイデンティティ・トーン・禁止表現・用語ルールを全エージェントが参照するための統一マニュアル。
このスキルを参照することで、あらゆる出力が「Seep Logos らしい」一貫したブランドを体現するようになる。

---

## 1. ブランドアイデンティティ

### ミッション
> **無限の"世界"を創生する**

### ブランドパーソナリティ
- **クリエイティブ**: 独自の世界観を生み出す創造力
- **情熱的**: コンテンツへの深い愛と情熱
- **緻密**: 細部にまでこだわる職人精神
- **グローバル**: 世界に届けるIP創生への志

### ブランドポジション
「ゲームやアニメ等のエンタメ作品制作と世界観構築が得意なクリエイティブスタジオ」

---

## 2. ブランドボイス & トーン

### 基本方針
| 媒体 | 文体 | トーン |
|---|---|---|
| プレスリリース | です・ます調 | 正式・専門的 |
| 公式Web | です・ます調 | 丁寧・熱意あり |
| SNS（公式） | です・ます調基本（親しみやすさ可） | 熱量・共感重視 |
| 採用広報 | です・ます調 | 誠実・情熱的 |
| 社内文書 | です・ます調 / だ・である調可 | 明確・フレンドリー |
| クリエイター向け | です・ます調 | リスペクト・誠実 |

### 好む表現
- 「世界観」「IP創生」「クリエイターファースト」「没入型エンタメ」
- 「物語を紡ぐ」「世界を創る」「創生する」
- 「クリエイター」（作家・アーティスト・デザイナー等の総称として）
- 「エンタメ作品」（ゲーム・アニメ・マンガ等の総称として）

### 避ける表現
- 「普通の」「一般的な」「よくある」→ Seep Logos の独自性を損なう
- 「弊社程度では」「まだまだ未熟ですが」→ 過度な謙遜
- 「業界初」「日本最大」「世界最高」→ 根拠のない誇大表現
- 「革命的な」「パラダイムシフト」→ 根拠のない大仰な表現

---

## 3. 禁止表現リスト

### 絶対禁止
- 競合他社の具体的な名指し批判
- 未発表IP・タイトルの名称（機密情報）
- 個人情報・未公開の人事情報
- 著作権侵害に当たる引用・転載

### 社外向け文書で禁止
- 社内プロジェクトコード名（正式タイトル発表前）
- 未確定の発売日・リリース時期の断言
- 未発表キャスト・クリエイターの氏名
- 社内の売上・財務情報

### SNS・プロモーション文書で禁止
- 絵文字・顔文字（公式SNS以外）
- 「いいね！」「RT！」等の強要的な表現
- ネガティブキャンペーン（「〇〇より面白い」等の比較）

---

## 4. 固有名詞・表記ルール

### 会社名
- 正式表記：「株式会社Seep Logos」（法的文書・プレスリリース）
- 通常表記：「Seep Logos」
- 略称：禁止（「SL」等の略称は使用しない）

### 一般表記ルール
- 数字：アラビア数字（1, 2, 3）を基本。ただし文頭は漢数字可
- 英数字：半角（全角英数字は原則禁止）
- カタカナ：外来語は原則カタカナ表記（例：ゲーム、アニメ、シナリオ）
- 括弧：「 」（黒括弧）を基本。( ) は補足説明に限る

### 作品関連用語
- ゲームタイトルは必ず正式表記で（略称・愛称は避ける）
- キャラクター名は設定資料の表記に従う
- 世界観固有名詞は用語集の表記に従う

---

## 5. ビジュアルブランドルール（概要）

### カラーの方針
- 使用カラーは各IP・プロダクトごとにアートディレクターが定義
- Seep Logos コーポレートカラーは別途ブランドブックを参照

### フォントの方針
- 社外向け日本語文書：游ゴシック・ヒラギノ等の清潔感ある書体
- ゲーム内フォントは各タイトルのデザインガイドを参照

### ロゴ使用ルール
- ロゴの改変・回転・伸縮は禁止
- 最小表示サイズは高さ20px以上
- 背景色により白抜き・黒抜きバージョンを使い分ける

## 6. よくあるミス & NG例

### NG例1：過度な謙遜
```
NG: 「まだまだ小さな会社ですが、精一杯頑張ります」
OK: 「Seep Logos は、世界観構築への情熱を持ったクリエイティブスタジオです」
```

### NG例2：根拠なき誇大表現
```
NG: 「業界革命を起こす、史上最高の没入型RPG」
OK: 「独自の世界観構築を強みとした、高没入型RPG」
```

### NG例3：競合批判
```
NG: 「〇〇社の作品と違い、私たちは本物の世界観を作っています」
OK: 「Seep Logos ならではの世界観構築アプローチで制作しています」
```

### NG例4：未確定情報の断言
```
NG: 「2025年夏リリース予定です」（未確定の場合）
OK: 「リリース時期は近日公開予定です」
```
