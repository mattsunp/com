# アートディレクションマニュアル（Art Direction）

## このスキルの目的
Seep Logos のIP・プロダクト全体のビジュアル方針策定・スタイルガイド管理・アートワーク品質管理の標準を定めたマニュアル。
「世界観が視覚から伝わる」クリエイティブを一貫して作るための指針。

---

## 1. アートディレクションの基本原則

### 世界観ビジュアル化の使命
ビジュアルは世界観の「証拠」である。
「世界観設定資料に書かれたこと」が、絵を見た瞬間に感じ取れるビジュアル設計を目指す。

### 一貫性と多様性のバランス
全媒体（ゲーム・グッズ・SNS・印刷物）で「同じ世界観を感じる」一貫性を保ちながら、媒体ごとの表現の最適化も行う。

### クリエイターへのリスペクト
外部イラストレーター・デザイナーへの指示は「制約」ではなく「世界観への招待」として伝える。
クリエイターの個性が発揮できる余白を残したディレクションを行う。

---

## 2. ビジュアルコンセプト設計プロセス

### Step 1: 世界観設定の視覚化
worldbuilding-creator が作成した設定資料を読み込み、以下を言語化する：
- 光の質（冷たい・温かい / 鋭い・柔らかい）
- 色の印象（彩度・明度・温冷色の傾向）
- テクスチャ感（荒い・滑らか / 古い・新しい）
- 時代感・世界観の「空気」

### Step 2: カラーパレット定義
```
【カラーパレット定義書の構成】
- メインカラー（1〜2色）: IP・作品の顔となる色
- サブカラー（2〜3色）: アクセントと補助
- ニュートラルカラー（1〜2色）: 背景・テキスト用
- 禁止カラー: 世界観に合わない色
- 感情マッピング: 各色が表現する感情・場面
```

### Step 3: スタイルガイド作成
全クリエイターが参照するビジュアルルールブック。含める要素：
- キャラクターデザインのルール（デフォルメ度・頭身・塗りのタッチ）
- 背景・環境のルール（遠近感・光源・描き込み量）
- フォントルール（タイトルロゴ・本文・UIテキスト）
- ロゴ・ブランドマークの使用ルール

---

## 3. ビジュアルブリーフの作成標準

### ビジュアルブリーフの必須要素
1. **案件概要**: 何のためのビジュアルか（媒体・用途・サイズ）
2. **コンセプトワード**: 5〜10語で表現（例：「廃れた黄金」「冬の静寂」「生命の爆発」）
3. **カラー方針**: 使う色・避ける色
4. **参照方向性**: 言葉で表現するビジュアルイメージ
5. **禁止要素**: 世界観上存在しないもの・ブランドNG要素
6. **合格基準**: 何をもって「合格」とするか

### 外部クリエイターへの依頼ポイント
- 「〇〇のような絵にして」ではなく「この作品の空気感は〜で、そこに住む人々は〜という感情を持っている」と世界観から伝える
- NG例を先に伝えることで認識のズレを防ぐ
- 初稿はラフ確認を必ず設ける（いきなり完成稿は避ける）

---

## 4. ビジュアル監修の基準

### OK基準（全て満たすこと）
- [ ] 世界観設定と整合している（技術レベル・時代感・文化）
- [ ] アートスタイルガイドのカラーパレットに準拠している
- [ ] キャラクターの個性・設定が視覚的に表現されている
- [ ] ターゲットユーザーに訴求する魅力がある
- [ ] 他媒体（グッズ・Web等）で使用しても世界観が保たれる

### フィードバックの書き方
```
NG: 「なんか違う」「もっと良くして」
OK: 「キャラクターの目の鋭さが設定（冷静沈着な性格）と合っていません。
    瞳の形を細くし、感情を表に出さない表情に変更してください」

NG: 「色が暗い」
OK: 「このシーンは希望の場面なので、現在の暗い色調（S30,B40程度）を
    もう少し明るく（S40,B60程度）調整してください」
```

---

## 5. 媒体別ビジュアル最適化

| 媒体 | 主な注意点 |
|---|---|
| ゲーム内UI | 情報の可読性・状態変化の明確さ |
| ゲーム背景 | キャラクターとのコントラスト・ゲームプレイの邪魔をしないか |
| SNS用ビジュアル | モバイル小画面での視認性・サムネイル映え |
| 印刷物（フライヤー等） | CMYK変換・解像度（350dpi以上） |
| グッズ | 商品素材・加工技術の制約 |

---

## 6. よくあるミス & NG例

### NG例1：ブリーフが抽象的すぎる
```
NG: 「かっこよくて、でも可愛い感じで、明るくて、ダークな感じもあって」
OK: 「昼の賑やかな市場と、夜の危険な裏路地が共存する街。
    豊かさの裏に腐敗が隠れる、矛盾に満ちた美しさを表現してください」
```

### NG例2：修正指示が感覚的すぎる
```
NG: 「全体的にもっとSeep Logosっぽく」
OK: スタイルガイドの具体的なページ・項目を指定して修正依頼する
```
