| 本編 | 千夏の案件（4幕構造・完全スコア） | 60〜75分 |
| 合計 | | 65〜85分 |

3件連鎖案は棄却。谷口綾の「倫理的判断体験」は千夏の案件の第1幕（依頼受諾）に移植。

### コアループの確定版

```
[プロローグ：操作習得]
断片整合パズル（3枚）+ 署名のみ
改竄なし / 感情整合なし / スコアなし
端末ブート演出 → 依頼受諾 → 作業 → 署名 → 終了
所要：5〜10分

[本編：5段階サイクル × 4幕]
スキャン → 整合 → 発見（マーキング） → 補完（感情整合） → アウトプット（署名 or 拒否）
感情整合スコア最大88〜92
所要：60〜75分
```

---

### 操作3種の確定仕様

| 操作 | 仕様 | 実装方針 |
|---|---|---|
| 断片整合パズル | ドラッグ&ドロップ式。配置確定でロック演出（カードが「嵌まる」音とアンバー文字の安定化） | 標準UI実装 |
| 改竄マーキング | テキスト範囲選択 + マーキングツール適用。「正解」があるが完璧でなくても進行可能 | 標準テキスト選択実装 |
| 感情整合パズル | 候補断片の選択（択一式）。感情属性タグによるスコア算出。リアルタイムスコア表示 | 属性タグデータ設計が必要 |

---

### 音響設計の確定方針

| シーン | 音響 |
|---|---|
| プロローグ全体 | 環境音のみ（BGMなし）|
| 本編・第1〜3幕 | 微細なアンビエントBGM（単音・非旋律系）|
| 書店シーン（千夏の断片） | 環境音のみ（BGMなし）← プロローグとの意図的な共鳴 |
| 第4幕・結末 | BGM停止。環境音のみ |

「無音楽」がプロローグと書店シーンと結末に共通することで、
「この瞬間だけが素の時間だ」というメッセージが音響設計として機能する。

### UIコンセプトの確定

- 全操作をゲーム内端末画面（レトロコンピューター風アンバーモニター）で完結
- ゲームUIをゲーム内物理オブジェクトとして描画（インゲームUI方針、v4から継続）
- 感情強度フラグ付き断片：微細ノイズ＋文字微振動（Unityシェーダーカスタム）
- 改竄痕跡：文字の二重写り（消した文章の残影）
- 端末ブート演出：ゲーム開始時のOSブートログシーケンス
