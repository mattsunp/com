## 最終世界観（世界観構築人）

**舞台：記憶の街「ムネーメー」**

現代と少しずれた都市。「記憶外科」という職業が存在し、消したい記憶を抱えた人が訪れるクリニックが街のそこかしこにある。

**世界のルール（設定三原則）：**

1. **記憶は物質として観測できる** — 専用機器「ニューロスコープ」を使えば、記憶は「光の残像」として視覚化される。鮮明なものは白く輝き、古びたものは黄ばみ、痛みを帯びたものは赤黒く滲む。

2. **忘却には副作用がある（連鎖消去）** — 記憶同士は「感情の共鳴」によって線（エッジ）で繋がっている。ニューロスコープで観測すると、依頼人の記憶空間はネットワーク図として可視化される。消去するノードに接続された「連鎖消去リスクライン」が赤く表示される。

3. **記憶は他者と共有できる** — 一部の人間は「刻印師（しるしし）」と呼ばれ、記憶の読み取り・移植が可能。主人公ナギはこの能力を持つ。

**記憶ネットワークの構造：**
- **ノード（点）：** 個別の記憶断片
- **エッジ（線）：** 感情的繋がり。太い線ほど連鎖影響が大きい
- **クラスター（群）：** 特定の感情・関係性を中心に形成された記憶の塊。核心ノードを消すとクラスター全体が揺らぐ

**街の断面：**
- 「消去ローン」— 辛い記憶を消すための費用を分割払いにできる金融サービス
- 「記憶銀行」— 大切な記憶を外部に預けるサービス。預けた記憶は本人から切り離される
- 「追憶喫茶」— 他人の記憶を「体験コンテンツ」として鑑賞できる店。倫理的グレーゾーン

**ムネーメーの倫理観：**
「記憶を消す権利は本人のもの」という法律がある。しかし「刻印師の判断が結果に影響する」という事実は公式には認められていない。刻印師は「補助者」であり「決定者」ではないことになっている——実態は違うが。

**主人公：ナギ**
記憶外科クリニック「クロニクル」に勤める若手の刻印師。彼女自身の記憶には大きな欠落がある——なぜかは本人も知らない。

## 最終物語骨格（シナリオ書き）

**三幕構成：**

**第一幕「刻印」**
依頼人・オルオが来る——「妻との10年間の記憶を全て消してほしい」。妻・ミドリは3ヶ月前に事故死。生き続けるために、忘れる必要があると言う。ナギは記憶探索に入るが、ミドリの「最後のメッセージ」を発見する。オルオ本人も知らないメッセージ。

**第二幕「剥離」**
ナギはオルオの記憶の奥へ踏み込む。オルオと妻の関係が「幸福ではなかった」事実が浮かぶ——しかし二人は確かに愛していた。「幸福でない愛」は消すべきか。ナギの欠落した記憶が、オルオの記憶の何かとリンクしていることに気づく。

**第三幕「忘却の許可」（最終改訂版）**
核心が明かされる——ミドリは以前、ナギのクリニックに来ていた。ミドリが消したかった記憶：「オルオが仕事を辞めて夢を諦めた理由が自分（ミドリ）への愛情だった」という事実。ミドリはその記憶を自分から消すことで、オルオが「自分のために諦めた」という負債感から解放されることを望んでいた。

ミドリはナギに言っていた——「彼に話してはいけない。ただ、消えた後に彼が前へ進めるよう、手伝ってほしい」。ナギはその言葉を処理した後、義務として忘れていた。

**三つの選択肢（最終版）：**
- （A）オルオの記憶を消す。ミドリの最後の望み通りに。オルオは前へ進む。真実は消える。
- （B）最後のメッセージだけ残す。「忘れていい」というミドリの言葉だけを残す。オルオは知らずに許される。
- （C）全てをオルオに渡す。ミドリが何を、なぜ消したかったかを含めて。オルオ自身が決める。

**いずれのエンディングでも：** ナギは「ミドリの依頼を処理した刻印師だった」という事実を思い出す。そして気づく——自分がその記憶を消したのか、消されたのか、まだ分からない、と。

**エピローグ：** ナギは一人、ニューロスコープで自分自身の記憶空間を開く。ネットワーク図に、無数の切断されたエッジが浮かぶ。エンドロール。

**感情動線：**
- 序盤：他人事の静けさ（職業的距離感）——プレイヤーは「判断者」
- 中盤：記憶への感情移入、ナギとの同期
- 終盤：「忘れていたのは誰か」という問いがプレイヤー自身に跳ね返る——プレイヤーは「共感者」

## 最終ゲームループ・体験設計（ゲームディレクター）

**確定ゲームループ：**

```
[依頼受諾]
→ [表層探索（白）：言語化された事実の収集]
→ [ニューロスコープで記憶ネットワーク把握]
→ [中層探索（黄）：感情・断片の収集]
→ [深層探索（赤黒）：ナギへの負荷発生、自身の記憶揺らぎ]
→ [感情マッピング完了：連鎖リスク確認]
→ [判断・選択]
→ [結果・ナギの記憶変容]
```

**UI設計確定仕様：**

1. **ニューロスコープUI：** 記憶のノードネットワーク図。消去対象ノードを選択すると連鎖リスクラインが赤く点灯する。プレイヤーはリスクを「見て」から判断する。

2. **セーブUI：** 記憶空間内ではセーブボタン・メニューを非表示にする。「記憶の中では外の時間が止まる」という世界観的説明を序盤に提示。バックグラウンドで自動保存。中断時は「ここで離れると記憶が不安定になる」という演出テキストを表示（実害なし、演出のみ）。

3. **テキスト変容UI：** 深層に踏み込むほど、過去の選択肢テキストが微妙に変化する。「削除した」→「消えていた」など。ナギの記憶が書き換わっていることをUIで表現する。

**プレイ設計：**
- 一本道に近い構造（分岐は第三幕の最終選択のみ）
- 探索の自由度は低く「記憶空間を歩く」感覚を優先
- 移動でなく「視点を動かす」操作系に統一
- プレイ時間：3〜4時間

**プレイ時間連動要素：** 「時間経過でUI輝度が落ちる」は第1作では見送り。代替として「チャプター間のロード画面に時間の経過を示す文章を1行表示する」軽量実装を採用。

## 各エージェントの論点と決着

| 論点 | 提起者 | 決着 |
|---|---|---|
| セーブ不可区間のリスク（モバイル・ストレス問題） | 企画人 | UIからセーブを消す＋バックグラウンド自動保存で解決。体験と実用を両立。 |
| 選択肢（C）の感情的リアリティ不足 | 世界観構築人 | 「ミドリがオルオを自由にするために記憶を消した」という動機を追加。忘却の許可がテーマの核心に。 |
| プレイヤーを共感者と判断者のどちらに設計するか | シナリオ書き | 二段階設計を採用。判断者として始まり共感者として終わる。没入のピークを終盤に設計。 |
| 忘却の副作用ルールの曖昧さ | ゲームディレクター | 記憶ノードネットワーク＋連鎖リスク表示で解決。設定がゲームメカニクスとして機能するように統合。 |
| プレイ時間連動UI（輝度変化）の実装コスト | ゲームディレクター（自己提起） | 第1作は見送り。チャプター間の1行テキストで代替。 |

## 全体戦略家（chief-producer）総括

LEHTEは「記憶と忘却」というテーマに対して、ゲームである必然性を確立した企画だ。
