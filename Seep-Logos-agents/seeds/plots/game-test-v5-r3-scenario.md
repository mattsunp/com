### 第1幕：依頼の受諾（改訂版）

**神崎航の登場と問い返しの設計**

v4では第1幕は「依頼の受付」として単純に設計されていた。
v5では「受けるかどうかの不確実性」を冒頭から埋め込む。

**シーン：事務所。午後。雨の音。**

```
神崎：「記憶修復士の……柳沢さん、ですか」

レイ：「はい。どうぞ」

神崎：「妻の、記憶データの整理と復元を——お願いしたいのですが」

　彼は「整理と復元」と言った。
　「整理」と「復元」は、本来、別の依頼だ。
　レイはそのことに気づいたが、黙っていた。

レイ：「一つだけ聞かせてください。
　　　 あなたは、妻の記憶の中で何を見たいのですか。
　　　 確認ですか。それとも——何かを、探しているのですか」

神崎：（沈黙。3秒）

神崎：「……それが、わかれば、ここには来ません」

　レイは端末に手を置いた。
　この仕事を受けるかどうか、選択するのはプレイヤーだ。

[選択：受ける / 断る]
```

**「断る」を選んだ場合**：

```
レイ：「申し訳ありませんが、今は受けられません」

　神崎は頷いた。何も言わずに立ち上がった。
　ドアが閉まる音。雨は続いていた。

　レイは端末の画面を閉じた。
　空白のままのファイルが一つ、残った。

　——翌日、同じ男が来た。
　今度は「依頼」ではなく「相談」として。
```

「翌日の再訪」によって、断ってもゲームが進行する。ただし「断った事実」がプレイヤーの記憶に残る。
この蓄積が第4幕の選択に影響する（ゲーム的な分岐ではなく、プレイヤーの感情的文脈として）。

### 第2幕：書店シーン（v5最終版）

**シーン：記憶断片「2054-09-13 / 14:22 / 感情強度フラグなし / MED非推奨空間」**

```
[端末画面の表示]

記録場所：Shimokitazawa / 古書店（MED非推奨空間）
記録時間：22分43秒
感情強度フラグ：なし
備考：MED非推奨空間での記録。非装着状態での記録ではない。

[再生]

　棚の端に、背表紙の色が褪せた本が一冊立っている。
　手を伸ばすと、隣の棚の老人と目が合った。

老人：「その本、面白いですよ。続きが気になって眠れなくなる」

千夏：「そうですか。それは困るな」

老人：「困りますかね。眠れなくなるのも、悪くないと思うが」

千夏：笑う。「……そうかもしれない」

　しばらく立ち読みをする。老人はいなくなっていた。
　本を買う。外に出ると、曇っていた。

　急がなくていい。待つ人はいる。でも、急がなくていい。

[記録終了]

　音楽はない。
　レイは画面を見ていた。
```

**シナリオの設計意図**：
「急がなくていい。待つ人はいる。でも、急がなくていい。」
この繰り返しは内声として書かれている。記録の外側にある思考だ。
MEDが「感情強度フラグなし」と判定した状態での言葉。
静かさが、彼女が生きていた証明になる。

### 第4幕：3分岐の結末（v5最終版）

**分岐A（元データで提出）**：
```
レイ：修復レポートに署名する。
　　  改竄の痕跡を記録として残す。

　　「残すことが、唯一の証言かもしれない」
　　　そう思ったかどうか、レイは口にしなかった。
```

**分岐B（追加データで提出）**：
```
レイ：最善の整合を選んで署名する。

　　「最善と真実が違うことは、知っている」
　　　それでも、最善を渡す。
　　　これが仕事だと思って。
```

**分岐C（破棄）**：
```
レイ：署名しない。ファイルを閉じる。
　　  書店のシーンだけを印刷する。
　　  壁の「紙の記録」に、一枚加える。

　「記録されなかったことが、消えたわけじゃない」
　　壁はすこし、賑やかになった。
```
