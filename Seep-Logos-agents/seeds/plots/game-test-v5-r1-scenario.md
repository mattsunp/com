## シナリオ書きの起点：「選択」の重みをどう設計するか

v4で確立された物語の骨格は優れている。
しかし「3件連鎖」構造を採用するとき、シナリオ設計で以下の問題が生まれる。

**問題：プレイヤーの感情的疲弊**

45〜75分の体験で1件の案件（千夏）に集中するv4の設計では、
感情的なエネルギーは1点に集中する。

3件連鎖になると、プレイヤーの感情的エネルギーが分散する。
第1件・第2件で消耗した状態で第3件（千夏）に到達する可能性がある。

これは欠陥ではない。「仕事の蓄積が判断を鈍らせる」という体験の一部にできる。
ただし意図的に設計しなければ「疲れた」という印象で終わる。

---

## 3件連鎖のシナリオ構造（提案）

### 感情カーブの設計

```
[第1件：谷口達也] — 感情温度：低
記録の確認作業。改竄なし。プレイヤーは「記憶修復士の仕事」を学ぶ。
終わり方：「ありがとう」の1文だけ。報酬の受け取り演出。淡白に終わる。
感情的余韻：なし（意図的に）

[第2件：谷口綾] — 感情温度：中
「依頼を受けるかどうか」の選択がある。受けた場合のみ内容に進む。
レイの「仕事の定義」が問われる最初の場面。
終わり方：どちらの選択でも「問いが残る」構造。答えが出ない。
感情的余韻：中程度のもやつき

[第3件：神崎航] — 感情温度：高（千夏の案件）
プレイヤーは「2件分の判断の蓄積」を持ってここに来る。
結末の3分岐で「蓄積された判断パターン」が浮き彫りになる。
感情的余韻：大
```

### 第2件の「依頼を受けるかどうか」の選択肢設計

この選択はゲームの世界に「倫理的判断」を持ち込む最初の瞬間。
重要なのは「どちらが正しいか」を示さないことだ。

**選択A：受ける**
「あなたにはその権利がある」とレイは言わない。
「やってみます」とだけ言う。
内容に入り、谷口達也の記憶の「父の顔」が現れる。

**選択B：断る**
「規則上は問題ない」とだけ言う。綾は「そうですか」と去る。
この後、第3件の神崎の依頼が来る。
神崎の依頼を「受ける」ことの対比として、断った理由をプレイヤーが再考する。

## 千夏の「書店シーン」のシナリオ詳細

世界観構築人が「書店＝非記録の空間」という設定を加えた。
これによってこのシーンのシナリオが一段深まる。

**シーン：記憶断片「2054-09-13 / 14:22 / 感情強度フラグなし」**

---

```
[表示：端末画面 / ファイルカード]

記録場所：Shimokitazawa / 古書店
記録時間：22分43秒
感情強度フラグ：なし
備考：MED非推奨空間での記録

[再生]

棚の端に、背表紙の色が褪せた本が一冊立っている。
手を伸ばすと、隣の棚の老人と目が合った。

老人：「その本、面白いですよ。続きが気になって眠れなくなる」

千夏：「そうですか。それは困るな」

老人：「困りますかね。眠れなくなるのも、悪くないと思うが」

千夏：笑う。「……そうかもしれない」

　しばらく立ち読みをする。老人はいなくなっていた。
　本を買う。外に出ると、曇っていた。

　急がなくていい。待つ人はいる。でも、急がなくていい。

[終了]
```

---

このシーンには「劇的な出来事」は何もない。
でも千夏は「急がなくていい」と思っている。
この1文が、彼女が生きていたことの証明だ。

**シナリオ設計の意図**：
「待つ人はいる」は神崎の存在を示唆するが、確定はしない。
「でも、急がなくていい」はそれでも彼女が「今ここ」にいることの表明。

このシーンは4幕の第2幕に配置する。
プレイヤーが「千夏＝データ」から「千夏＝人」に切り替わる瞬間だ。

## シナリオ書きからの問いかけ（Round 2への引き継ぎ）

1. 第1件の「淡白な終わり方」は、プレイ体験として正解か。ゲームの冒頭が感情的に平坦であることのリスクをゲームUI設計で補えるか
2. 書店シーンの「MED非推奨空間での記録」という表記は、世界観の説明なしにプレイヤーに伝わるか。チュートリアルとの連携が必要か
3. 第2件「断る」選択をしたプレイヤーに対して、第3件の神崎への「受ける」選択の対比を強調する演出が必要か
