## コンセプト（最終）

**タイトル**：『PALIMPSEST — 記憶の修復士 —』（暫定）
**ジャンル**：パズル × ナラティブ（一人称・端末インターフェース型）
**プレイ時間**：65〜85分（プロローグ5〜10分 + 本編60〜75分）
**プレイ人数**：1人
**推奨開発エンジン**：Unity（URP）

**ターゲット**：
- メイン：20〜35歳、論理思考を好むゲーマー。脱出ゲーム・ポイントアンドクリック系の愛好者
- サブ：ミステリー小説・SF小説を好む読書家層（非ゲーマー含む）

**コアループ**：
「記録の断片を読み解き、削除・改竄を発見し、感情的に最も整合する配置を判断する」

**操作3種**：
| 操作 | 内容 | 行為の意味 |
|---|---|---|
| 断片整合パズル | 記憶断片（ファイルカード）をドラッグ式で時系列・感情軸に配置 | 他者の記憶の流れを「理解する」 |
| 改竄マーキング | テキスト選択で矛盾・感情異常値を名指し | 嘘に名前をつける |
| 感情整合パズル | 空白に「感情的に最も整合する断片」を選択。スコア可視化（最大88〜92） | 他者の記憶の空白を「最善の解釈」で埋める |

## 世界観（最終）

**舞台**：2055年、日本・東京近郊

**社会背景**：
2040年代に「記憶外部化デバイス（MED）」が普及。普及率65〜70%。
強度感情状態での記録には「感情強度フラグ」が付与される。
「観察効果の恒常化」——MEDの普及により「記録されている意識」が常態化し、自然な発言が失われつつある社会。

**書店という場所の象徴性**：
書店・図書館は「MED非推奨空間」として「素の言葉が交わされる場所」になっている。
千夏が書店で老人と話した記録は「感情強度フラグなし・MED非推奨空間での記録」という特殊な注記を持つ。
千夏はその場でMEDをオフにしていなかった。この事実をゲームは解釈しない。

**記憶修復士という職業**：
認定資格制。署名が法的効力を持つ。
「記憶改竄の被害者側の復元・鑑定」を請け負う民間職。
法廷用の証言より「自分のための記録の回復」を求める依頼人が多い。

**柳沢 レイ（34歳・女性）**：
思想的選択者としてMED非装着。
過去に記憶の証言が誰かを傷つけた経験を持つ（詳細は語らない）。
古い端末と旧規格ドライブが積み重なる事務所。壁一面に「紙の記録」——過去の依頼者から受け取った手書きの言葉たち。

## 物語の骨格（最終）

**登場人物**：
- 柳沢 レイ（34歳・女性）：記憶修復士。プレイヤーが操作する視点人物
- 神崎 航（42歳・男性）：大手製薬会社開発部長。妻の記憶復元を依頼する
- 神崎 千夏（39歳・女性・故人）：3ヶ月前に死亡。残した記憶データがこのゲームの「舞台」となる
- 谷口 達也（48歳・男性）：プロローグの依頼人。チュートリアル用

**全体構成**：

| 構成 | 内容 | 時間 |
|---|---|---|
| プロローグ | 谷口達也の案件。改竄なし・感情整合なし。「空白はそのままにする」体験 | 5〜10分 |
| 第1幕：依頼の受諾 | 神崎の来訪。「何を見たいのか」というレイの問い返し。受諾 or 断るの選択 | 10分 |
| 第2幕：発見と転換 | 欠落パターンの発見。書店シーン（千夏が「人」になる瞬間） | 20分 |
| 第3幕：矛盾と補完 | 追加データの登場。感情整合パズルで「どちらの千夏が本当か」を問われる | 25分 |
| 第4幕：選択と結末 | 修復レポートの作成。3分岐 | 10分 |

**体験の核心**：
書店の老人と話す千夏のシーン——感情強度フラグなし、特別なドラマなし——が、
プレイヤーの中で千夏を「データ」から「人」に転換させる。
3つの結末のどれを選んでも、プレイヤーはこの千夏と別れる。

**3分岐の結末**：
- A（元データで提出）：改竄の痕跡を記録として残す。「残すことが唯一の証言」
- B（追加データで提出）：最善の整合を選ぶ。「最善と真実が違うことは知っている」
- C（破棄）：書店のシーンだけを印刷して壁の紙の記録に加える。「記録されなかったことが、消えたわけじゃない」

**神崎の来訪シーンの核心セリフ**：
```
レイ：「一つだけ聞かせてください。
　　　 あなたは、妻の記憶の中で何を見たいのですか。
　　　 確認ですか。それとも——何かを、探しているのですか」

神崎：「……それが、わかれば、ここには来ません」
```

## ゲームプレイ設計（最終）※v5追加

**UIコンセプト**：
全操作をゲーム内端末画面（レトロコンピューター風アンバーモニター）で完結。
インゲームUI方針（UIをゲーム内の物理オブジェクトとして描画）。

**視覚表現**：
- 通常データ：クリーンなアンバー文字
- 感情強度フラグ付き断片：微細ノイズ＋文字の微振動
- 改竄痕跡：文字の二重写り（消した文章の残影）
- 端末ブート演出：ゲーム開始時のOSブートログシーケンス

**音響設計**：
| シーン | 音響 |
|---|---|
| プロローグ全体 | 環境音のみ（BGMなし） |
| 本編・第1〜3幕 | 微細なアンビエントBGM（非旋律系） |
| 書店シーン | 環境音のみ（BGMなし）← プロローグと意図的に共鳴 |
| 第4幕・結末 | BGM停止、環境音のみ |

「無音楽」がプロローグ・書店シーン・結末の3点に共通することで、
「この瞬間だけが素の時間だ」というメッセージが音響として機能する。

**感情属性タグシステム**：
各断片に `calm / grief / tension / expectation / confusion / warmth` の6タグを付与。
隣接断片の属性遷移の相性でスコアを算出。最大スコアは88〜92に設定。
「完璧な整合（100点）は存在しない」という事実をデータ設計で保証する。

**開発フェーズ（暫定）**：
| フェーズ | 内容 | 目安工数 |
|---|---|---|
| Phase 0（プリプロ） | データ設計・シェーダープロトタイプ | 2〜3週間 |
| Phase 1（Vertical Slice） | 第2幕1断片セット実装 | 4〜6週間 |
| Phase 2（α版） | 全4幕実装 | 8〜12週間 |

## エージェント間の論点と決着

### 論点A〜F（v4から継続）
→ v4最終版に記録済み。v5では継承。

### 論点G：プレイ時間の定義
**争点**：3件連鎖で90〜120分に膨張するか。「短編」の定義はどこか。
**決着**：プロローグ（5〜10分）＋本編（60〜75分）= 65〜85分に設計変更。短編の範囲内と判断。

### 論点H：神崎の「悪意の明示度」の再検討
**争点**：シナリオ書きが設計した「何かを探している」という神崎の姿勢が、悪意を早期に示しすぎるか。
**決着**：「それがわかれば、ここには来ません」という返答は「悪意の明示」ではなく「不確実性の提示」として設計する。プレイヤーは序盤から「この人物が何者か判断できない」状態に置かれる。これはテーマと整合する。

### 論点I：書店シーンの音響設計
**争点**：「書店シーンのみ音楽なし」をプレイヤーがどこで理解するか。
**決着**：チュートリアル（プロローグ）全体も「音楽なし・環境音のみ」で設計することで、「無音楽＝素の時間」という法則をプレイヤーが自然に習得する。書店シーンが同じ質感で現れたとき、説明なしに「ここは違う」と感じる。全員同意。

## 全体戦略家所見

判断する。

v5は「ゲームとして動く」設計の欠落を正確に補完した。
v4が「企画・世界観・シナリオの3本柱」で立っていたとすれば、
v5では「実装設計」という4本目が加わり、企画全体が立体になった。

**特に評価する点**：

「音響設計の統一性」——プロローグ・書店シーン・結末を「無音楽」でつなぐという発見は、
ゲーム開発人の参加なしには生まれなかった。
この3点の「素の時間」が通奏低音として機能することで、
千夏の存在感は「音楽がないこと」によって担保される設計になった。
Seep Logosの「没入感」原則に対して、本v5の最大の貢献はここにある。

**動的スタッフィングの評価**：

ゲーム開発人を「Round 1完了後に追加招集」したことで、
不必要に早い段階での実装制約をコンセプト設計に持ち込まずに済んだ。
企画・世界観・シナリオが自由に議論を進めた後に、実装視点が加わる順序が正しかった。
これは今後の企画プロセスのモデルとして記録する。
