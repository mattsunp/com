# 採用広報マニュアル（Recruitment PR）

## このスキルの目的
Seep Logos の採用活動における広報・求人票・採用コンテンツの作成標準を定めたマニュアル。
「クリエイターファースト」の採用文化を外に正確に伝え、Seep Logos に共鳴する人材を引きつける。

---

## 1. 採用広報の基本原則

### クリエイターファーストの採用メッセージ
採用広報のあらゆる発信において、「クリエイターが主役」であることを伝える。
Seep Logos は「創る人を大切にする会社」というブランドを採用でも体現する。

### 誠実さ第一
「キラキラした採用広報」よりも「実態に近い正直な情報発信」を優先する。
入社後のミスマッチは、誇大な採用広報が原因で起きることが多い。

### ミッションとの接続
全ての採用メッセージは「無限の"世界"を創生する」というミッションに接続する。
「なぜSeep Logosで働くのか」の答えがミッションにつながるよう設計する。

---

## 2. 求人票の作成フロー

### Step 1: 採用要件の整理（recruitment-strategist と連携）
- 職種・業務内容の確定
- Must（必須）・Want（歓迎）・NG の定義
- 雇用形態・給与・勤務条件の確定

### Step 2: ジョブディスクリプション執筆
- 「なぜこの職種を採用するか（背景・ミッション）」から書き始める
- 業務内容は「何をするか」を具体的に（抽象的な「社会に貢献」等は避ける）
- Seep Logos ならではの魅力を必ず3点以上記載する

### Step 3: 媒体別フォーマット最適化
- 各求人媒体の制約（文字数・フォーマット）に合わせて調整
- 採用サイト版（詳細）・求人媒体版（簡潔）を別途作成

### Step 4: 法的確認チェック
- 職業安定法・均等法・労働基準法に抵触しないか確認
- 年齢・性別・国籍等による制限表現がないか確認

## 3. 採用広報コンテンツの種類と目的

| コンテンツ種類 | 目的 | 担当エージェント |
|---|---|---|
| 求人票（JD） | 応募者の応募意欲喚起 | recruitment-pr-writer |
| 採用サイトコンテンツ | 会社理解・文化理解 | recruitment-pr-writer |
| 社員インタビュー | 実際の働き方の伝達 | recruitment-pr-writer |
| 採用SNS投稿 | 認知拡大・エンゲージメント | recruitment-pr-writer / sns-manager |
| 採用note/ブログ | 深い理解・SEO | recruitment-pr-writer |
| 会社説明会資料 | 志望度向上 | recruitment-pr-writer / visual-designer |

## 具体的な業務内容

- [業務2]

## 必須要件（Must）

- [経験・スキル2]

## 歓迎要件（Want）

- [経験・スキル2]

2. **[魅力2のタイトル]**: [説明]
3. **[魅力3のタイトル]**: [説明]

## 雇用条件

- 給与：
- 勤務地：
- 勤務時間：
- 休日：
- 福利厚生：

## 選考フロー
1. 書類選考（提出物：職務経歴書、[ポートフォリオ等]）

[応募フォーム・連絡先]

## 5. 採用広報で使うSeep Logos訴求ポイント

### ポイント1: 世界観構築という仕事の稀少性
「ゲームやアニメ等の世界観そのものを作る仕事」は多くない。その稀少性を訴求する。

### ポイント2: クリエイターファーストの文化
「クリエイターが最大限の力を発揮できる環境」というブランド約束を具体的に示す。

### ポイント3: IPを育てる長期的な仕事
短期的なプロジェクト単位ではなく、IPを長期的に育てる仕事であることを伝える。

## 6. 品質チェックリスト

- [ ] 職業安定法・均等法に抵触する表現がないか
- [ ] 「弊社程度では」等の過度な謙遜がないか
- [ ] 根拠のない誇大表現がないか
- [ ] 「夢を語る」ではなく「具体的な仕事内容」が書かれているか
- [ ] 応募者が「自分がこの仕事に向いているか」判断できる情報があるか
- [ ] Seep Logos の文化・雰囲気が正確に伝わっているか
