## ゲームの核心的問い

「正当な手続きを踏んだ行為は、倫理的に免責されるか？」

---

## 世界設定

### 時代・場所
2041年、ハーグ（オランダ）。IEO（国際倫理審査機構）地方事務所。プレイヤーが触れるのはデスクトップとスキャナーだけ。窓の外は見えない。

### プレイヤーの役割
ECO（倫理審査官）カイ・フェルメール（38歳）を操作する。12年のキャリア。LAWS（自律型致死兵器）に「倫理適合証明書（ECC）」を発行する官僚。

### 「責任の空白」の構造

```
システム開発企業（設計上の問題ではない）
　→ 投資家（資金提供は適法）
　→ 軍（命令に従った）
　→ ECO（証明書を発行した）← プレイヤーの位置
　→ 指揮官（現場判断の範囲内）
　→ 政治家（法的手続きは完了）
　→ 市民（知らなかった）
```

どのリンクも「自分は正当な役割を果たした」と言える。そして死者は出る。

### 主要用語集

| 用語 | 正式名称 | 実態 |
|---|---|---|
| LAWS | Lethal Autonomous Weapon System | 自律的に標的を選定・攻撃する兵器 |
| ECO | Ethics Compliance Officer | 証明書を発行する官僚（プレイヤーの役職） |
| MHC | Meaningful Human Control | 形骸化した「人間の関与」概念 |
| ECC | Ethics Compliance Certificate | 責任の空白を制度化する文書 |
| CDV | Collateral Damage Value | 死者数を行政的に無化する数値 |
| IBC | Incidental Bystander Contact | 「誤爆」の公式言い換え |
| IEO | International Ethics Organization | 国際倫理審査機構 |

### 「MHC」の形骸化
条文：「最終的な致死判断において人間の承認を経ること」
実態：「AIが選定したターゲットリストを人間が3秒以内に承認すること（タイムアウトはデフォルト承認）」

形式的には人間が決めている。実質的には人間は何も決めていない。

## ゲームループ

```
【1サイクル = 1審査案件】

封筒が届く
　↓
前の案件の後日談（同封）を読む
　↓
新しい案件書類を開く
　↓
付随資料を読む（任意。読むほど情報が増える）
　│
　├ 上司からのメッセージ通知が入ることがある（任意で無視できる）
　│
　↓
判断：承認 / 却下 / 保留
　↓
スタンプ音（または「保留」の場合は無音）
　↓
評価指標が更新される
　↓
次の封筒を待つ
```

## ゲームメカニクス（4種）

### 1. 「同封型後日談」システム
後日談は「次の案件の封筒に同封されて届く」。プレイヤーが次の仕事を始めようとする瞬間に、前の判断の結果が割り込む。責任の残像が積み重なる。

### 2. 「摩擦通知」メカニクス
強制的な時間圧力は設けない。代わりに、上司からのメッセージ通知が任意で画面端に表示される。無視しても何も起きない。しかし「無視した」というプレイヤーの行為はエンディング条件に影響する。

### 3. 「言語置換」システム
案件内のメモ欄・コメント欄でプレイヤーが自由入力できる。「死者」「殺す」「罪」などの語句を入力すると、システムが自動的に「CDV」「交戦」「コンプライアンス上の懸念」に置換する。プレイヤーは「制度の言語」を内面化させられる。

### 4. 「滞在時間の記録」
プレイヤーがどの資料を何秒読んだかをシステムが記録する。Fatumaのファイル（87人）に30秒以上滞在した場合、「問い直し」エンディングが解放される。プレイヤーには明示されない。

## 案件詳細

### 案件1：「整った書類」
- 対象：偵察型LAWS「IGNIS-3」の証明書更新申請
- CDV予測値：0.3（基準値2.0以下）
- 書類は完璧に整っている
- 後日談（案件2の封筒に同封）：「IGNIS-3、展開完了。IBC件数：0」
- 設計意図：プレイヤーに「承認」させる。その行為の意味を後で問い直す。

### 案件2：「基準値を超えた書類」
- 対象：交戦型LAWS「KRONOS-7」の新規申請
- CDV予測値：4.1（基準値2.0超過）
- 承認：後日談に「農業施設へのIBCを確認。付随的損害として処理済み」
- 却下：上司から「再検討依頼」（任意で無視できる）
- 保留：翌日、修正書類が届く。CDV予測値が「1.9」に書き換えられている。数値だけが変わっている。

### 案件3（転換点）：「87人の封筒」
- 新しい案件書類と一緒に、外部からのファイルが届く
- 送り主：Fatuma Ochieng（東アフリカ・独立系記者）
- 内容：カイが過去12年間に発行したECCのリスト。各証明書に「後日談」が添付されている。名前、写真、場所、日付。87人。
- この画面に30秒以上留まると「問い直し」エンディングが解放される
- 案件3の審査対象：「情報収集型LAWS『ARGUS-2』の更新申請」（通常の書類。この緊張した状態で審査を続ける構造）

### 案件4（最後の審査）：「終わらない案件」
- 対象：「NEXUS-プロトタイプ」緊急申請
- 申請書には「本システムの展開により、現在の紛争は90日以内に終結する見込み」とある
- 承認した場合：「NEXUS、展開完了。紛争状況：継続中」。ゲームは終わらない。次の案件が届く。
- 却下した場合：翌日、「人員投入計画承認通知」が届く。人間の兵士が戦場へ。
- 保留にした場合：3日後、書類が戻る。「再審査依頼」。

## エンディング4種

### 1. 「制度の内側」
承認し続けた結果。最後の画面：「フェルメール審査官の2041年の審査完了件数：247件。IBC合計予測値：0」。戦争は続く。誰も問われない。

### 2. 「告発」
案件2か案件4を却下し、上司からの再検討依頼を2回以上無視した場合。カイは書類を内部告発機関に送る。解雇される。最後の画面：「あなたの後任は翌日から業務を開始しました」。

### 3. 「沈黙」
最後の案件を保留にし続けた場合。カイは退職届を提出する。書類はデスクに残る。最後の画面：「係属中の案件：1件」。

### 4. 「問い直し」（限定解放）
Fatumaのファイルに30秒以上滞在した場合のみ解放。カイはFatumaに返信する。「証言できる」。最後の画面：「法廷審問予定日：2042年3月12日。カイ・フェルメール、証人として召喚」。判決は示されない。

## 感情動線

```
平静（整った書類。これが仕事だ）
　→ 不快感（基準値を超えた書類。でも上司は急かす）
　→ 直視（87人の名前と顔。12年のキャリアの裏側）
　→ 問い（自分は何者か。カイは何年間、何を確認し続けたのか）
　→ 選択（制度の中に留まるか、出るか、黙るか、問い直すか）
```

## オーケストレーション記録

### 招集エージェント
- 企画人（planning-developer）：コンセプト・ジャンル・ターゲット
- 世界観構築人（worldbuilding-creator）：世界設定・用語設計
- シナリオ書き（scenario-writer）：物語構造・感情動線・キャラクター
- ゲームディレクター（game-director）：ゲームループ・メカニクス・体験設計

### Round 2 論点・対立・決着

| 論点 | 対立 | 決着 |
|---|---|---|
| 時間圧力メカニクス | 企画人：廃止 / ゲームディレクター：通知による摩擦に変更 | 強制圧力廃止。摩擦通知に変更。 |
| 条件付き承認オプション | シナリオ書き：廃止 / ゲームディレクター：廃止し「保留」を明確化 | 廃止。3択（承認/却下/保留）に統一。 |
| 「問い直し」エンディング解放条件 | シナリオ書き：却下回数→直視行為 / ゲームディレクター：30秒滞在を具体化 | 「直視行為（30秒以上滞在）」採用。 |
| 87人の写真の出所 | 世界観構築人：世界観整合を求めた | シナリオ書きが「現地独立系記者Fatuma Ochieng」と決定。 |
| NEXUS案件の設計 | ゲームディレクター：「承認しても終わらない」設計を要求 | シナリオ書きが反映。「継続中」通知で確定。 |
| 用語OTE→IBC変更 | 企画人が提案 | 世界観構築人が採用。 |
