# 新作アニメ発表 X投稿テンプレート集

> 対象：新作アニメの初動発表フェーズ（ティザー〜発表当日〜発表直後）
> ブランドルール適用済み｜sns-manager作成

---

## 凡例

```
{{タイトル}}        ：作品タイトル
{{放送時期}}        ：20XX年X月放送予定 / 20XX年放送予定 等
{{放送局/配信}}     ：テレビ東京ほか / Netflix / Amazon Prime Video 等
{{スタジオ名}}      ：制作スタジオ名
{{公式サイトURL}}   ：https://...
{{ハッシュタグ}}    ：#タイトル名 等の作品固有タグ
{{世界観コンセプト}}：作品のコアコンセプトを1文で
```

## 運用メモ

### ハッシュタグ設計方針

| 用途 | 例 |
|---|---|
| 作品固有タグ | `#{{タイトル}}` |
| Seep Logos統一タグ | `#SeepLogos` |
| カテゴリタグ | `#新作アニメ` `#アニメ新作情報` `#アニメ化決定` |
| シーズンタグ | `#2026春アニメ` `#2026夏アニメ` 等 |
| 原作連動タグ | `#{{原作タイトル}}` `#{{漫画タイトル}}アニメ化` 等 |
| キャスト関連タグ | `#{{タイトル}}キャスト` |

### 投稿の基本ルール

- 絵文字は1投稿あたり0〜4個を上限とする
- 情報カテゴリの区切りに絵文字を使い、装飾目的に多用しない
- 文体は「です・ます」調を基本とする
- 「普通」「一般的」「よくある」等の表現は使用しない
- 「革命的」「業界初」等の根拠なき誇大表現は使用しない
- ハッシュタグは投稿末尾にまとめる（本文内への埋め込みは最小限）
