# 新作アニメ発表 X投稿テンプレート集（汎用版）

> 対象：新作アニメの初動発表フェーズ（ティザー〜制作決定〜放送開始〜最終回）
> 対応プラットフォーム：X（旧Twitter）専用テンプレート
> Instagram・TikTok等への展開は別途プラットフォーム別テンプレートを参照すること
> 汎用版（2026-03-25改訂）：業界標準の文言パターンに基づく｜sns-manager作成

---

## 設計方針

本テンプレートは、アニメ業界の公式SNSアカウントで広く使われている文言パターンを調査・分析した上で設計している。独自表現を排し、プレースホルダーを差し替えるだけで即時運用できる汎用性を最優先としている。

---

## 凡例

```
[作品タイトル]    ：アニメの正式タイトル
[原作タイトル]    ：原作ある場合の原作名
[放送時期]        ：2026年X月放送予定 / 2026年春放送予定 等
[放送クール]      ：2026年春 / 2026年秋 等
[放送局/配信]     ：TOKYO MX・BS11ほか / Netflix / Amazon Prime Video 等
[制作スタジオ]    ：制作スタジオ名
[公式サイトURL]   ：https://...
[ハッシュタグ]    ：#作品タイトル 等の作品固有タグ
```

## よく使われる定型フレーズ一覧

調査で確認された業界頻出フレーズ。以下から状況に応じて選択・組み合わせて使用する。

## 運用メモ

### ハッシュタグ設計方針

| 用途 | 例 |
|---|---|
| 作品固有タグ | `#[作品タイトル]` |
| カテゴリタグ | `#新作アニメ` `#アニメ新作情報` `#アニメ化決定` |
| シーズンタグ | `#[放送クール]アニメ`（例：`#2026春アニメ`） |
| キャスト関連タグ | `#[作品タイトル]キャスト` |
| 原作連動タグ | `#[原作タイトル]` `#[原作タイトル]アニメ化` |
| イベント・感想タグ | `#[作品タイトル][話数]話`（例：`#〇〇1話`） |

### 文体・フォーマット共通ルール

- 文体は「です・ます」調を基本とする
- タイトル見出しには「【】」を使用する（業界標準）
- 情報の区切りには「■」または「▼」を使用する
- ハッシュタグは投稿末尾にまとめる
- 絵文字は0〜3個を目安とし、多用しない
- 「！」は感情を強調する場面で使用する（1投稿あたり1〜3個程度）
- 「普通」「一般的」「よくある」等の表現は使用しない
- 「革命的」「業界初」等の根拠なき誇大表現は使用しない
- 本文内の情報区切りには「▼」を使用する

### 文字数目安

- X（旧Twitter）の日本語投稿は全角140文字が上限（半角は2文字で1カウント）
- 本テンプレートの変数展開後の概算：
  - ティザー期（Phase 1）：30〜70字 + ハッシュタグ20〜40字
  - 発表本番（Phase 2・3）：60〜130字 + URL（約25字換算）+ ハッシュタグ20〜40字
  - 各話放送後（Phase 4）：30〜80字 + ハッシュタグ20〜40字
  - キャンペーン（Phase 6）：100〜140字（URL・ハッシュタグ含む）

### ビジュアル仕様メモ

- 制作決定・ティザー期：16:9横長（推奨 1200×675px）または 1:1正方形
- キービジュアル公開：縦長 1080×1350px も可
- PV公開時：サムネイル画像を1枚添付し、動画URLはテキスト内に記載する
- 複数枚添付の場合は最大4枚まで
- ビジュアル制作時は visual-designer（意匠人）との仕様確認を必ず実施すること
