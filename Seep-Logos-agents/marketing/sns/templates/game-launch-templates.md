# 新作ゲーム発表 X投稿テンプレート集（汎用版）

> 対象：新作ゲームの初動発表フェーズ（ティザー〜発表当日〜リリース日）
> 対応プラットフォーム：X（旧Twitter）専用テンプレート
> Instagram・TikTok等への展開は別途プラットフォーム別テンプレートを参照すること
> 汎用版（2026-03-25改訂）：業界標準の文言パターンに基づく｜sns-manager作成

---

## 設計方針

本テンプレートは、ゲーム・アニメ業界の公式SNSアカウントで広く使われている文言パターンを調査・分析した上で設計している。独自表現を排し、プレースホルダーを差し替えるだけで即時運用できる汎用性を最優先としている。

### ⚠️ 投稿設計の最重要原則：「現実」と「架空世界」のセパレート

1投稿内に**世界観パート（架空世界側）** と **告知パート（現実側）** が共存することは許容する。ただし、両者が地続きで溶け合わないよう、**文中で境界を明確にセパレートすること**。

**NG（境界なし）：**
```
傘は、雨粒から守るためだけにあるわけじゃない。続報をお待ちください。
```

**OK（空行による余白）：**
```
傘は、雨粒から守るためだけにあるわけじゃない。

新作ゲーム『Umbrella』近日公開。続報は公式アカウントでお知らせします。
```

> 区切り線は使わない。空行で間を作る。線は主張するが、空白はただそこにある。受け手はその余白の中で自然に切り替わる。

## 凡例

```
[作品タイトル]    ：ゲームの正式タイトル
[ジャンル]        ：RPG / アクション / ADV / ストラテジー 等
[プラットフォーム]：PS5 / Nintendo Switch / PC(Steam) / iOS・Android 等
[発売時期]        ：2026年X月発売予定 / 2026年発売予定 等
[公式サイトURL]   ：https://...
[ハッシュタグ]    ：#作品タイトル 等の作品固有タグ
[会社名]          ：運営・パブリッシャー名（省略可）
```

## よく使われる定型フレーズ一覧

調査で確認された業界頻出フレーズ。以下から状況に応じて選択・組み合わせて使用する。

### 情報公開・解禁系
- 「情報解禁！」「情報解禁しました」
- 「ティザーサイトを公開しました」「公式サイトを公開しました」
- 「PVを公開しました」「ティザーPVを公開しました」
- 「ティザービジュアルを公開しました」「キービジュアルを公開しました」
- 「本日より〇〇を公開しています」

### 告知・お知らせ系
- 「〇〇の開発・発売が決定しました」
- 「〇〇についてお知らせいたします」
- 「〇〇を発表いたします」
- 「新作ゲーム『[作品タイトル]』の発売が決定いたしました」

### 期日・日程明示系
- 「[発売時期]発売予定」
- 「本日より」「本日[発売時期]よりサービス開始」
- 「[日付]（[曜日]）[時刻]より」
- 「[日付]リリース予定」

### 事前登録・キャンペーン系
- 「事前登録受付中！」「事前登録受付中です」
- 「事前登録者数[X]万人突破！」
- 「事前登録特典として〇〇をプレゼント」
- 「事前登録はこちら▼」

### リリース・サービス開始系
- 「本日サービス開始！」「本日より配信スタートです！」
- 「ついにリリース！」「本日ついにリリースとなりました！」
- 「App Store・Google Playにて配信中！」
- 「ダウンロードはこちら▼」「今すぐプレイ▼」

### 感謝・報告系
- 「多くの方にご支援・ご声援をいただき、ありがとうございます」
- 「おかげさまで事前登録者数[X]万人を達成しました」
- 「たくさんのご反響をいただきありがとうございます」
- 「引き続きよろしくお願いいたします」

### 続報予告系
- ⚠️ 「続報をお待ちください」は世界観への没入を阻害するメタ発言。エンターテインメント作品のプロモーションでは原則使用しないこと。代替表現（下記）を使用する。
- 「続報は公式アカウントでお知らせします」（事実の告知として許容）
- 「近日、詳細をお知らせします」
- 「詳細は公式サイトをご確認ください▼」
- 「フォローして最新情報をチェック！」
- 「次回の情報公開をお楽しみに！」

## 運用メモ

### ハッシュタグ設計方針

| 用途 | 例 |
|---|---|
| 作品固有タグ | `#[作品タイトル]` |
| カテゴリタグ | `#新作ゲーム` `#ゲーム新作情報` `#インディーゲーム` |
| ジャンルタグ | `#RPG` `#アクションゲーム` `#ADV` `#ストラテジー` 等 |
| プラットフォームタグ | `#PS5` `#NintendoSwitch` `#Steam` `#スマホゲーム` 等 |
| イベントタグ | `#[作品タイトル]リリース` `#[作品タイトル]発売` `#キャンペーン` 等 |
| シーズンタグ | `#2026年発売` `#2026夏発売` 等 |

### 文体・フォーマット共通ルール

- 文体は「です・ます」調を基本とする
- タイトル見出しには「【】」を使用する（業界標準）
- 情報の区切りには「■」または「▼」を使用する
- ハッシュタグは投稿末尾にまとめる
- 絵文字は0〜3個を目安とし、多用しない
- 「！」は感情を強調する場面で使用する（1投稿あたり1〜3個程度）
- 「普通」「一般的」「よくある」等の表現は使用しない
- 「革命的」「業界初」等の根拠なき誇大表現は使用しない

### 文字数目安

- X（旧Twitter）の日本語投稿は全角140文字が上限（半角は2文字で1カウント）
- 本テンプレートの変数展開後の概算：
  - ティザー期（Phase 1）：30〜70字 + ハッシュタグ20〜40字
  - 発表本番（Phase 2）：60〜120字 + URL（約25字換算）+ ハッシュタグ20〜40字
  - キャンペーン告知（Phase 5）：100〜140字（URL・ハッシュタグ含む）

### ビジュアル仕様メモ

- ティザー期：16:9横長（推奨 1200×675px）または 1:1正方形
- 発表本番：キービジュアル全体（縦長 1080×1350px も可）
- 複数枚添付の場合は最大4枚まで
- PV公開時：サムネイル画像を1枚添付し、動画URLはテキスト内に記載する
