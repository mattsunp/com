# アニメ SNS投稿スケジュール（全フェーズ版）

> 対象：新作TVアニメ（深夜帯・配信同時・地上波放送）の制作発表以降から放送終了後まで
> sns-manager作成｜ブランドルール適用済み
>
> **注意：制作発表前のティザー期〜正式発表当日のスケジュールは `announcement-schedule.md` を参照すること。**
> **本資料はその後続、すなわち「制作発表以降〜放送終了後フェーズ」を扱う。**

---

## 全体タイムライン

```
【フェーズ0】制作発表前（ティザー）
  → announcement-schedule.md を参照

【フェーズ1】制作発表後〜放送3ヶ月前
  世界観開拓期：週2〜3回。キャスト・スタッフ・世界観の核心を届ける

【フェーズ2】放送3ヶ月前〜放送1ヶ月前
  加速期：週3〜5回。PV・場面カット・放送局情報と連動

【フェーズ3】放送1ヶ月前〜放送開始
  クライマックス期：週5回〜毎日。カウントダウン・最終情報解禁

【フェーズ4】放送中
  週次定常運用：放送曜日を軸に固定ルーティンで運用

【フェーズ5】最終回後〜放送終了後
  余韻醸成・次展開準備期：週2〜3回
```

---

## フェーズ1｜制作発表後〜放送3ヶ月前（世界観開拓期）

### フェーズ概要

| 項目 | 内容 |
|---|---|
| 投稿頻度 | 週2〜3回 |
| 投稿時間 | 20:00〜22:00 優先 |
| 推奨曜日 | 水曜・木曜・日曜（エンゲージメントが安定する傾向） |

### 週次ローテーション（コンテンツ種別）

| 週 | 投稿内容 | メディア |
|---|---|---|
| 第1週 | キャスト第1弾発表（主人公・ヒロイン等主要役） | キャラ設定画＋キャスト名 |
| 第2週 | スタッフ情報発表（監督・シリーズ構成・キャラデ） | スタッフビジュアル or テキスト |
| 第3週 | キャラクター第2弾（サブキャラ・ライバル） | 設定画 |
| 第4週 | 世界観・ストーリー第1弾（あらすじ・舞台） | コンセプトアート or ティザービジュアル |
| 第5〜8週 | キャスト追加・世界観深掘りをローテーション継続 | — |

### コンテンツ設計の指針

- **キャスト発表**：キャラクターの魅力と声優のコメントをセットにすることで、声優ファン層への波及を狙う
- **スタッフ情報**：監督・シリーズ構成等の過去作品との接続性を文脈として添えることで「期待値の根拠」を作る
- **世界観投稿**：「この世界で生きるとはどういうことか」が伝わる断片を届ける。設定の全量を開示せず、想像力の余白を残す

## フェーズ2｜放送3ヶ月前〜放送1ヶ月前（加速期）

### フェーズ概要

| 項目 | 内容 |
|---|---|
| 投稿頻度 | 週3〜5回 |
| 投稿時間 | 20:00〜22:00 優先 |
| 推奨曜日 | 月・水・金・日（週4回体制の場合） |

### このフェーズで発信すべき重点コンテンツ

| 施策 | タイミング | 内容 |
|---|---|---|
| PV第1弾公開 | 放送90日前前後 | 世界観と音楽を軸にした「雰囲気重視」の映像 |
| 放送局・配信サービス発表 | 放送80日前前後 | どこで・いつ見られるかを明確に告知 |
| PV第2弾公開 | 放送60日前前後 | キャラクターの動き・ストーリーの断片を追加 |
| 主題歌・楽曲情報解禁 | 放送50日前前後 | OP/ED情報＋アーティスト（歌手）へのクロス告知 |
| 全キャスト情報解禁 | 放送40日前前後 | キャスト一覧ビジュアル等でまとめて公開 |
| 各話話数・各話タイトル情報 | 放送30日前前後（任意） | 話数が確定しているなら公開 |
| キャスト陣コメント映像 | 放送前月 | 期待感の最大化とファンとの距離感縮小 |

## フェーズ3｜放送1ヶ月前〜放送開始（クライマックス期）

### フェーズ概要

| 項目 | 内容 |
|---|---|
| 投稿頻度 | 週5回〜毎日 |
| 投稿時間 | 20:00〜22:00 最優先 |

### カウントダウン投稿設計

| タイミング | 内容 |
|---|---|
| 放送30日前 | 「あと1ヶ月」告知＋全キャスト集合ビジュアル |
| 放送14日前 | 「あと2週間」＋ローンチPV公開 or 先行場面カット |
| 放送7日前 | 「あと1週間」＋第1話あらすじ・先行カット |
| 放送3日前 | 「あと3日」＋視聴方法（局・配信サービス）の最終案内 |
| 放送前日 | 「明日放送開始」＋放送時刻・推奨ハッシュタグの告知 |
| 放送当日 | 朝「本日放送開始」宣言 → 放送直前リマインド → 放送後「ありがとう」投稿 |

### 放送当日（第1話）の投稿時刻設計

| 時刻 | 内容 |
|---|---|
| 8:00〜9:00 | 「本日第1話放送です」宣言 |
| 放送1時間前 | 放送時刻リマインド＋推奨ハッシュタグ告知 |
| 放送直後（〜30分以内） | 「第1話放送ありがとうございました」感謝投稿 |
| 放送当日深夜〜翌日 | 先行場面カット公開（ネタバレに配慮しつつ印象的なシーン） |

## フェーズ4｜放送中（週次定常運用）

### 放送中の基本姿勢

放送中のアニメSNSは「リアルタイム視聴体験をSNSで増幅する」ことが最大の役割。
X(Twitter)ユーザーの78%がテレビ視聴中にXを同時利用しているというデータがあり、
放送と連動したリアルタイム投稿設計が最もエンゲージメントを高める。

### 週次投稿ルーティン（固定フォーマット）

> ※ 放送曜日が毎週木曜深夜の場合を例示する。曜日は作品に合わせて読み替えること。

| 曜日 | 投稿内容 | 投稿時刻 | 目的 |
|---|---|---|---|
| 月曜 | 先行場面カット公開（今週放送分） | 20:00 | 週後半の放送への期待醸成 |
| 水曜 | 次回話数タイトル＋あらすじ告知 | 20:00 | 視聴予約・リマインドの促進 |
| 木曜（放送当日） | 放送1時間前リマインド → 放送直後感謝投稿 | 放送1H前 / 放送直後 | リアルタイム視聴体験の強化 |
| 金曜 | 放送回の振り返り投稿・感想ハッシュタグ紹介 | 20:00〜21:00 | ファンコミュニティの盛り上がりを公式が後押し |
| 日曜 | 週次コンテンツ（キャラ深掘り・スタッフコメント等） | 20:00 | 週末エンゲージメント維持 |

### 各投稿の設計詳細

#### 先行場面カット公開（月曜）
- ネタバレになりすぎない印象的なシーンを1〜3枚
- 「今週は〇〇が鍵になります」等の一言コメントで興味を引く
- ハッシュタグ：公式タグ＋話数タグ（例：#作品名 #第X話）

#### 次回予告・あらすじ（水曜）
- 話数タイトル＋30〜50字のあらすじ
- 次回映像の埋め込みがあれば添付
- 「次回もお楽しみに」の定型文は避け、固有の見どころを一言添える

#### 放送当日リマインド（放送1時間前）
- 放送チャンネル・時刻・配信サービスを明記
- 推奨ハッシュタグを案内し、ファンが感想を集約しやすくする
- 「ぜひリアルタイムで」の声かけ

#### 放送直後の感謝投稿（放送終了〜30分以内）
- 「第X話ご視聴ありがとうございました」という定型文をベースに、各話固有のメッセージを1文添える
- 見逃し配信サービスURLを掲載
- 感想ハッシュタグへの誘導

#### 振り返り投稿（金曜）
- 公式が選んだ「今週の1シーン」をハイライト
- 「みなさんの感想で特に印象的だったもの」を引用（ファンへのリスペクトを示す）
- 次回放送への期待をつなぐ一言

### 特別投稿タイミング（放送中イレギュラー対応）

| タイミング | 対応内容 |
|---|---|
| 中盤（5〜6話放送後） | 「折り返し」投稿。前半の総括＋後半への期待告知 |
| 最終回1週前 | 「次回最終回」宣言。事前のドキドキ感を共有 |
| 最終回当日 | 放送前リマインド → 放送直後感謝 → 翌日総括 の3点セット |

## フェーズ5｜放送終了後（余韻醸成・次展開準備期）

### フェーズ概要

| 項目 | 内容 |
|---|---|
| 投稿頻度 | 週2〜3回（急に投稿が止まることでフォロワーを失わないよう維持） |
| 投稿時間 | 20:00〜22:00 |

### 終了直後（最終回翌日〜2週間）

| タイミング | 内容 |
|---|---|
| 最終回翌日 | 全話放送完走のお礼＋総括メッセージ |
| 最終回翌週 | スタッフ・キャストからのコメント映像 or 文章 |
| 〜2週間以内 | Blu-ray/DVD・配信サービスの告知 |

### 終了後の継続コンテンツ（2週間〜）

| コンテンツ種別 | 内容 | 頻度目安 |
|---|---|---|
| メモリアル投稿 | 「放送X周年」「あの名シーン」の振り返り | 節目ごと |
| 映像ソフト・配信情報 | Blu-ray発売日・特典内容・配信追加情報 | 確定次第随時 |
| イベント告知 | 舞台挨拶・声優イベント・ライブ | 確定次第随時 |
| 続編・関連作品発表 | 続編決定・スピンオフ・コラボ情報 | 確定次第随時 |
| ファンアート紹介 | 公式がピックアップして紹介 | 月2〜4回 |

## 投稿頻度のフェーズ別まとめ

| フェーズ | 期間目安 | X(Twitter) | Instagram | YouTube |
|---|---|---|---|---|
| 世界観開拓期 | 制作発表後〜3ヶ月前 | 週2〜3回 | 週1〜2回 | 月1〜2回（PV・設定動画） |
| 加速期 | 3ヶ月前〜1ヶ月前 | 週3〜5回 | 週2〜3回 | 月2〜3回 |
| クライマックス期 | 1ヶ月前〜放送開始 | 毎日〜週5回 | 週3〜4回 | 週1〜2回（PV・特報） |
| 放送中 | 放送期間全体 | 週4〜5回（放送日連動） | 週2〜3回 | 週1回（次回予告・振り返り） |
| 放送終了後 | 終了後2ヶ月以上 | 週2〜3回 | 週1〜2回 | 月1〜2回（BD/DVD告知等） |

## ハッシュタグ戦略

### 基本ハッシュタグ設計

| カテゴリ | 例 | 用途 |
|---|---|---|
| 公式タグ | #作品名 | 全投稿に必ず添付。ファンの感想集約地点に |
| 話数タグ | #作品名_第X話 | 各話放送日に使用。リアルタイム盛り上がりの核 |
| イベントタグ | #作品名_X周年 / #作品名_BD | キャンペーン・記念投稿専用 |
| ジャンルタグ | #アニメ #今期アニメ #深夜アニメ | 新規ファン獲得のためのリーチ拡大 |

### 推奨ハッシュタグ数（X(Twitter)）

| フェーズ | 推奨タグ数 |
|---|---|
| 発表〜放送前 | 3〜5個 |
| 放送中（放送当日） | 2〜3個（公式タグ＋話数タグ中心に絞る） |
| 放送終了後 | 2〜4個 |

> 多すぎるハッシュタグはスパム感を生む。放送中はシンプルに絞ることが最も効果的。

## 炎上・品質リスクチェックリスト（投稿前確認）

- [ ] 場面カットに未放送エピソードのネタバレが含まれていないか
- [ ] キャストのコメントを誤引用していないか
- [ ] 放送局・配信サービスの情報が最新かつ正確か
- [ ] 視聴者の感想引用時に個人を特定・傷つける内容がないか
- [ ] 競合他社・他作品の名指し・過度な比較が含まれていないか
- [ ] 「最高傑作」「史上最高」等の根拠なき誇大表現がないか
