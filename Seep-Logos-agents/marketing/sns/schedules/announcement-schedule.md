# 新作発表 X投稿スケジュールガイド

> 対象：新作ゲーム・アニメの初動発表フェーズ全体
> sns-manager作成｜ブランドルール適用済み

---

## 基本方針

- **発表当日（T-0）を起点**として、前後のスケジュールを設計する
- ティザー期は「情報を小出しにして期待を高める」構造を持つ
- 発表後の続報は「週1〜2回」を基本ペースとし、情報過多を避ける
- 投稿時間帯は**18:00〜22:00（ゴールデンタイム）**を優先する
- 重大発表は**20:00前後**が最適（テレビニュースとの重複を避けつつ、視聴者が活発な時間帯）

---

## Phase別スケジュール（全体俯瞰）

```
T-21日   ティザー第1弾（キャッチコピー・謎めかし）
T-14日   ティザー第2弾（ビジュアル断片・設定ヒント）
T-7日    ティザー第3弾（ジャンル/放送媒体のヒント）
T-3日    完全ビジュアル公開 + 発表日告知
T-1日    前日予告
─────────────────────────────────────────
T-0日    ◆ 正式発表（午前：宣言 → 20:00：本発表 → 夜：補足）
─────────────────────────────────────────
T+1日    キャスト/スタッフ第1弾発表
T+3日    世界観・ストーリー深掘り
T+7日    放送/発売情報詳細 + 次回予告
T+14日   続報（キャラクター詳細・PV追加公開等）
T+30日   次フェーズへ（イベント・グッズ・放送開始カウントダウン等）
```

## 詳細スケジュール

### T-21日｜ティザー第1弾

| 項目 | 内容 |
|---|---|
| 投稿時間 | 20:00 |
| 投稿数 | 1投稿 |
| 内容 | キャッチコピーのみ（謎めかし型） |
| メディア | ティザー画像（タイトルなし）またはテキストのみ |
| ハッシュタグ数 | 2〜3個 |
| 目的 | アカウント注目度の引き上げ・期待醸成 |

---

### T-14日｜ティザー第2弾

| 項目 | 内容 |
|---|---|
| 投稿時間 | 20:00 |
| 投稿数 | 1投稿 |
| 内容 | キャラクタービジュアルの断片 or 世界観設定ヒント |
| メディア | 部分公開画像（クロップ・モザイク・暗転等）|
| ハッシュタグ数 | 3〜4個 |
| 目的 | ビジュアル面での関心喚起 |

---

### T-7日｜ティザー第3弾

| 項目 | 内容 |
|---|---|
| 投稿時間 | 20:00 |
| 投稿数 | 1〜2投稿（ゲーム：プラットフォームのヒント、アニメ：放送媒体のヒント）|
| 内容 | ジャンル・媒体・時期のヒント |
| メディア | テキスト中心 or 設定画断片 |
| ハッシュタグ数 | 3〜4個 |
| 目的 | 「いつ・どこで見られるか」への関心喚起 |

---

### T-3日｜完全ビジュアル公開

| 項目 | 内容 |
|---|---|
| 投稿時間 | 18:00〜20:00 |
| 投稿数 | 1投稿 |
| 内容 | キービジュアル全公開 + 発表日告知 |
| メディア | 高品質画像（1200×675px推奨） |
| ハッシュタグ数 | 4〜5個 |
| 目的 | 発表当日への期待の最大化 |

---

### T-1日｜前日予告

| 項目 | 内容 |
|---|---|
| 投稿時間 | 20:00 |
| 投稿数 | 1投稿 |
| 内容 | 明日発表・時間告知 |
| メディア | テキストのみ or ビジュアルリマインド |
| ハッシュタグ数 | 2〜3個 |
| 目的 | 翌日のアクセス予約・フォロー誘導 |

---

### T-0日｜◆ 正式発表当日

| タイミング | 投稿時間 | 内容 | 備考 |
|---|---|---|---|
| 朝 | 8:00〜9:00 | 「本日発表します」宣言 | 1投稿 |
| 発表本番 | 20:00 | タイトル・ビジュアル・基本情報の全公開 | **最重要投稿** |
| 補足① | 21:00〜22:00 | 公式サイト誘導 | 1投稿 |
| 補足② | 22:00〜23:00 | 反響感謝＋続報予告 | 1投稿（任意）|

**発表本番投稿のチェックリスト**
- [ ] タイトル正式表記の確認
- [ ] プラットフォーム/放送媒体情報の確認
- [ ] 発売時期/放送時期の確認
- [ ] 公式サイトURLの稼働確認
- [ ] ハッシュタグの表記確認
- [ ] 画像・動画ファイルの最終確認
- [ ] プレスリリースとの同時配信タイミングの確認

| 週 | 投稿内容の例 |

## ゲーム・アニメ別の差分ポイント

| 観点 | 新作ゲーム | 新作アニメ |
|---|---|---|
| 発表の呼称 | 「発表」「解禁」 | 「制作決定」「アニメ化決定」 |
| 続報の優先情報 | ゲームシステム・プラットフォーム | キャスト・スタッフ・放送局 |
| 発売/放送まで | 数ヶ月〜1年以上が多い | 3〜6ヶ月が多い |
| 追加メディア | ゲームプレイ映像（α/β版） | 各話予告・場面カット |
| ファン参加施策 | 体験版・早期予約特典告知 | 放送実況・応援上映告知 |

---

## 投稿時間帯の優先度

| 優先度 | 時間帯 | 理由 |
|---|---|---|
| 最高 | 20:00〜22:00 | ゲーム・アニメ層が最も活発な時間帯 |
| 高 | 18:00〜20:00 | 帰宅後〜夕食時 |
| 中 | 12:00〜13:00 | 昼休み |
| 低 | 8:00〜9:00 | 朝の宣言投稿（補助的）|
| 避ける | 1:00〜7:00 | 深夜〜早朝（リーチが著しく低下） |

---

## 投稿間隔の原則

- 同日複数投稿の場合：**最低4時間の間隔**を空ける
- 過剰投稿（1日5投稿超）は、フォロワーのミュート・フォロー解除を招くリスクがある
- 発表当日は例外的に3〜4投稿が許容される（ゴールデンタイム集中型）
