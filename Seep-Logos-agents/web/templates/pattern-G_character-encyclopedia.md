# ワイヤーフレーム：Pattern G「Character Encyclopedia型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：多キャラクター展開の作品。キャラへの感情移入・推しキャラ文化を活かした設計
- **ページ数**：1ページ（キャラ一覧）＋キャラ詳細（モーダル or 個別ページ）
- **必要最低素材**：ロゴ、キャラクタービジュアル（立ち絵/バスト）×3以上、キャラ名・肩書き
- **推奨スタイル**：グリッドレイアウト、ホバーエフェクト、キャラカラーを各カードに反映

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| HERO背景 | キービジュアル | ブランドカラーグラデーション |
| キャラ立ち絵/バスト | キャラクタービジュアル | シルエット＋「？」（謎演出） |
| キャラ名・肩書き | テキスト | 必須 |
| CV名 | キャスト情報 | 非表示 |
| プロフィールテキスト | 各100〜200字 | 「近日公開」テキスト |
| ボイスサンプル | 音声ファイル | 非表示 |
| ストーリービジュアル | キーシーン | カラーブロック |
| あらすじテキスト | 200〜300字 | 必須 |

## 拡張ロードマップ

1. キャラ追加ごとにグリッドカードを追加（シルエット→ビジュアル公開の演出も有効）
2. キャラ数が10人超：フィルタータブを活性化
3. 声優イベント・キャラクターグッズ情報をモーダルに追加
4. 長期運用：モーダルにキャラクターエピソード・ストーリーアーカイブを追加
