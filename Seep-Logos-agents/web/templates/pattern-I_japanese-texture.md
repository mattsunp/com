# ワイヤーフレーム：Pattern I「Japanese Texture型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：和風・時代劇・武士道・妖怪・神話などの和的世界観を持つ作品
- **ページ数**：1ページ（縦スクロール）
- **必要最低素材**：ロゴ（筆文字推奨）、和風テクスチャ素材 or ブランドカラー、世界観テキスト
- **推奨スタイル**：和紙テクスチャ・墨・筆文字・縦書き・金/朱/藍アクセント・余白多め

## 和風テクスチャ/カラーガイド

| 要素 | 推奨方針 |
|---|---|
| 背景 | 和紙テクスチャ（米色・薄墨色）またはダーク和風（深墨・夜空） |
| アクセントカラー | 朱色（#C0392B）/ 金色（#D4AF37）/ 藍色（#1B4F72）から1〜2色 |
| フォント | 游明朝・筑紫明朝・Noto Serif JP（本文）+ 筆文字フォント（見出し） |
| 装飾要素 | 家紋・波・流水・雲・桜・紅葉 など作品世界観に合わせて選択 |
| 罫線・区切り | 細い朱色 or 金色のライン。二重線も有効 |

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| HERO背景 | 和風キービジュアル | 和紙テクスチャ＋墨にじみ |
| タイトルロゴ | 筆文字ロゴ | 游明朝等のタイトルテキスト |
| 縦書きキャッチコピー | 1〜2文 | 必須 |
| 世界観ビジュアル | 水墨調・地図・風景 | 朱色/金色の幾何学模様 |
| 世界観テキスト（縦書き） | 200〜300字 | 必須 |
| キャラ立ち絵（和装） | 和装キャラクタービジュアル | シルエット |
| キーシーンビジュアル | 印象的な場面 | 和紙＋引用テキスト |

## 拡張ロードマップ

1. 海外向け展開時：縦書きを横書きに切り替え、英語テキスト対応（和風ビジュアルはそのまま活かせる）
2. キャラ追加：CHARACTERS グリッド拡張
3. PV公開：HERO下に動画セクション追加（Pattern F のTRAILERセクション流用）
4. 大型タイトル成長：Pattern Jへ移行
