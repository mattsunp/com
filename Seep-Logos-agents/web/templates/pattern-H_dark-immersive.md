# ワイヤーフレーム：Pattern H「Dark Immersive型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：SF・ディストピア・サイバーパンク・ダーク世界観の作品。ブランドカラーが暗色主体
- **ページ数**：1ページ（縦スクロール・フルダーク統一）
- **必要最低素材**：ロゴ、ダーク系キービジュアル、ブランドカラー（暗色＋アクセントカラー）
- **推奨スタイル**：黒・深紺基調、ネオンアクセント、グリッチエフェクト、ノイズテクスチャ

## ダークUIカラーガイド

| 要素 | 推奨カラー方針 |
|---|---|
| 背景 | #000000〜#0D0D0D（黒・超深色） |
| テキスト | #E0E0E0〜#FFFFFF（高コントラスト） |
| アクセント | 1色のみ集中（例：シアン・マゼンタ・電光系オレンジ） |
| セクション境界 | グラデーションフェード or 細いアクセントカラーライン |
| グリッチ/ノイズ | テクスチャオーバーレイ（透明度10〜20%） |
