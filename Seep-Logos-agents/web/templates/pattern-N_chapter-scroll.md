# ワイヤーフレーム：Pattern N「Chapter Scroll型」

作成日：2026-03-31
担当エージェント：デジタル・技術部 / web-director
ビジュアルレビュー：アートディレクション部 / art-director

## 概要
- **用途**：ストーリーや世界観テキストが先行している作品の紹介。絵本・ビジュアルノベル・小説原作IPの先行公開、ゲームや映像化前の世界観プレゼンテーションに最適
- **ページ数**：1ページ（縦スクロール・チャプター固定読了型）
- **必要最低素材**：タイトルテキスト、各チャプターのテキスト（200〜500字 × 3〜7章）、ブランドカラー（1〜2色）
- **推奨スタイル**：深みのある単色背景、セリフ体 × サンセリフ体の使い分け、余白重視、チャプター番号による章立て表現

---

## 差別化ポイント（対 A〜M）

| 軸 | 本パターンの立場 |
|---|---|
| vs Pattern E（Scrollytelling） | Eはスクロール演出（視差・アニメーション）が主役。Nはテキスト読解体験が主役。演出はテキストの補助にとどめる |
| vs Pattern B（World-first） | BはビジュアルでIP世界観を一撃で見せる。Nは言葉で世界を順番に「語りかける」 |
| vs Pattern H（Dark Immersive） | Hは視覚・音響の没入。Nは文学的没入。画像ゼロでも成立するのがN固有の特性 |
| vs Pattern M（Timeline Immersion） | Mは年表（時系列整理）。Nは物語進行（読者体験）。構造ではなく語りが主軸 |
| 独自性 | チャプター番号UI＋読了進行インジケーター＋テキスト中心レイアウトの組み合わせは A〜M に存在しない |

├─────────────────────────────────────────────────────────────────┤

## CHAPTER NAV（固定・右端・縦型）

```
              ┌────┐
              │  ○ │ ← PROLOGUE（通過済み：塗りつぶし）
              │    │
              │  ○ │ ← CHAPTER I（読中：パルスアニメーション）
              │    │
              │  ○ │ ← CHAPTER II
              │    │
              │  ○ │ ← CHAPTER III
              │    │
              │  ○ │ ← EPILOGUE
              └────┘
```

**要素メモ：**
- 右端固定・幅24px、各チャプターをドット（○）で表示
- スクロール位置に応じてアクティブドットがハイライト（塗りつぶし＋グロウ）
- 通過済みドットは通常の塗りつぶし色、未到達は枠線のみ
- ドットクリックで該当チャプターへスムーズスクロール
- ホバー時にチャプタータイトルがツールチップ表示
- モバイルでは非表示（代わりにスワイプ誘導のみ）

## PROLOGUE（フルスクリーン・100vw × 100vh）

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│      [PLACEHOLDER: タイトルロゴ]         │
│      ※ なし時：大きなタイトルテキスト   │
│      （セリフ体・中央揃え）              │
│                                         │
│                                         │
│      ─────────────────────              │
│                                         │
│      [PLACEHOLDER: 冒頭の一文]          │
│      例：「あの夜、世界は静かに終わった」│
│      ※ 1〜2文。必須。作品の声を最初に  │
│        聞かせる。                        │
│                                         │
│                                         │
│      ▼  続きを読む                       │
│                                         │
└─────────────────────────────────────────┘
```

**アニメーション：**
- タイトルが上からフェードイン（0.8s）
- 区切り線が左→右へ伸びる（0.4s delay）
- 冒頭の一文が下からフェードイン（0.6s delay）
- 「▼ 続きを読む」が穏やかな上下パルス
- 背景：単色ダークまたは紙テクスチャ風グラデーション（CSSのみ）

---

## CHAPTER（各章・ min-height: 100vh）

各チャプターは同じ構造。章が深まるごとに背景色を微妙に変化させる（暗→明、または逆）。

```
┌─────────────────────────────────────────┐
│                                         │
│  CHAPTER  I                             │
│  ─────────────────────                  │
│  [PLACEHOLDER: 章タイトル]              │
│  例：「滅びの前夜」                      │
│                                         │
│  ┌─────────────────────────────────┐    │
│  │                                 │    │
│  │  [PLACEHOLDER: 本文テキスト     │    │
│  │   200〜500字]                   │    │
│  │                                 │    │
│  │  ※ なし時：Lorem ipsum 相当の  │    │
│  │    プレースホルダーテキスト     │    │
│  │    （日本語）                   │    │
│  │                                 │    │
│  └─────────────────────────────────┘    │
│                                         │
│  [PLACEHOLDER: 引用・強調テキスト]       │
│  例：「──彼女の名前は、誰も知らない」    │
│  ※ 任意。1文。章の核心を抜き出して      │
│    大きく表示する（ブロッククオート風）  │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

**テキスト出現アニメーション：**
- ビューポートに入ったとき、テキストブロックが下から静かにフェードイン
- 章番号・タイトルは本文より先行して表示（0.2s lead）
- 引用テキストは本文より遅れて表示（0.4s delay）、フォントサイズを1.4倍に

**章ごとの背景色変化：**

| 章 | 背景色トーン | 意図 |
|---|---|---|
| PROLOGUE | 最も深い暗色（#0d0d0d） | 物語開幕の緊張感 |
| CHAPTER I | 暗色（#111111） | 世界の重さ |
| CHAPTER II | やや持ち上げた暗色（#161616） | 関係の深まり |
| CHAPTER III | 中間暗色（#1a1a1a） | 転換点 |
| CHAPTER IV（任意） | やや明るい暗色（#1e1e1e） | 解放への予感 |
| EPILOGUE | 最も明るい暗色（#222222） または明転 | 物語の余韻 |

※ ライトテーマ使用時は逆転（白→クリーム→薄いグレーへ）

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| タイトルロゴ | SVGロゴ・画像 | タイトルテキスト（セリフ体・大サイズ） |
| 冒頭の一文（PROLOGUE） | 作品を象徴する1〜2文 | 必須（最小必須素材） |
| 章タイトル | 各章の名称 | 必須（例：「第一章　滅びの前夜」） |
| 本文テキスト | 200〜500字の物語テキスト | 必須（これが主役） |
| 引用テキスト | 章の核心を抜き出した1文 | 任意（なくても成立） |
| 締めの言葉（EPILOGUE） | 余韻を残す1〜3文 | 必須 |
| 背景 | なし（CSSグラデーションで代替） | 不要 |
| イラスト・画像 | なし（テキストのみで完結） | 不要 |

## スクロール挙動仕様

| 操作 | 挙動 |
|---|---|
| マウスホイール | 標準縦スクロール |
| キーボード（↑↓） | 標準縦スクロール |
| キーボード（PageDown/Space） | 次章へジャンプ |
| キーボード（PageUp） | 前章へジャンプ |
| CHAPTER NAVドットクリック | 該当章へスムーズスクロール |
| タッチスワイプ | 標準縦スクロール |

## タイポグラフィ方針

| 用途 | フォント種別 | サイズ目安 |
|---|---|---|
| タイトル（PROLOGUE） | セリフ体（Georgia / 游明朝 / Noto Serif JP） | 3.5〜5rem |
| CHAPTER番号 | モノスペース / サンセリフ（細） | 0.75rem、大文字 |
| 章タイトル | セリフ体 | 1.8〜2.4rem |
| 本文テキスト | セリフ体 または 読みやすいサンセリフ | 1rem〜1.125rem、行間1.9〜2.1 |
| 引用テキスト | セリフ体・イタリック | 1.4〜1.6rem、中央揃え |
| CTAボタン | サンセリフ（細） | 0.875rem |

**アート管理者補足：**
本文テキストの行間は1.9以上を確保すること。詰めた行間は読解速度を落とし没入を妨げる。
引用テキストは必ず水平方向に余白（padding）を広くとり、「息を吸う空間」を設ける。
章番号は小文字禁止。ローマ数字（I・II・III）を推奨。

---

## 拡張ロードマップ

1. チャプター数は3〜7で調整可。テキスト量に合わせて増減
2. 各チャプターに環境音（小さな自然音・BGM）を割り当てると聴覚没入が加わる
3. 章ごとのシェアボタン（X・Instagram）を追加すると「読んだ」行動を促進
4. 本文テキストに「続きは作品本編で」の誘導リンクを自然に埋め込む（過度な促進は没入を壊すため最小限に）
5. 縦書きバージョン：日本語テキストを縦書き（writing-mode: vertical-rl）で表示するバリアント
6. 画像解禁後：各チャプターに差し替え可能な背景イラストスロットを追加（Pattern H との融合）
