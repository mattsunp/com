# ワイヤーフレーム：Pattern M「Timeline Immersion型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director
ビジュアルレビュー：アートディレクション部 / art-director

## 概要
- **用途**：世界観設定が厚い作品・シリーズIP。年表・歴史・時系列を旅させることで「その世界に生きている」感覚を与える
- **ページ数**：1ページ（横スクロール・タイムライン構造）
- **必要最低素材**：ロゴ、各時代のイベントテキスト（5〜10件）、ブランドカラー
- **推奨スタイル**：ディープネイビー×アンバーゴールド、グロウライン演出、セリフ体、慣性横スクロール

---

## 差別化ポイント（対 A〜L）

| 軸 | 本パターンの立場 |
|---|---|
| vs Pattern E（Scrollytelling） | Eは縦スクロールで「物語を語る」。Mは横スクロールで「歴史を旅する」 |
| vs Pattern G（Character Encyclopedia） | Gはキャラ図鑑（縦閲覧）。Mは時系列主軸でキャラ・事件を横断的に配置 |
| vs Pattern J（Hub-and-Spoke） | Jはリンク集。Mは没入体験優先・情報探索は副次的 |
| 独自性 | 横スクロール＋時代区分＋グロウライン演出の組み合わせは A〜L に存在しない |

## NAV（固定・縦型ミニマム）

```
┌──────┐
│[ロゴ]│  ← 左上固定
│      │
│ ERA  │  ← 各時代へのジャンプリンク（縦並び）
│  01  │
│  02  │
│  03  │
│  04  │
│  05  │
│      │
│[CTA] │  ← 登録/購入ボタン（最下部）
└──────┘
```

**要素メモ：**
- ナビは縦型サイドバー（左端固定・幅72px）
- 時代番号のクリックで該当ERAまでスムーズスクロール
- スクロール位置に応じて現在ERA番号がハイライト（アクティブ表示）

## HERO（フルスクリーン・100vw × 100vh）

```
┌─────────────────────────────────────────┐
│                                         │
│  [PLACEHOLDER: 背景ビジュアル]           │
│  ※ なし時：ディープネイビーグラデーション│
│                                         │
│                                         │
│      [PLACEHOLDER: タイトルロゴ]         │
│                                         │
│      [PLACEHOLDER: キャッチコピー 1〜2行]│
│      例：「すべての始まりを、旅せよ。」   │
│                                         │
│                                         │
│      ─ スクロールして時代を旅する ─      │
│         →→→  （右矢印アニメーション）   │
│                                         │
└─────────────────────────────────────────┘
```

**アニメーション：**
- ロゴがフェードインで出現
- キャッチコピーが下から滑り上がる（0.4s delay）
- 右矢印が左→右へ往復アニメーション（横スクロールへの誘導）
- HEROとTIMELINE STAGEは連続体として設計。スクロール開始でシームレスにタイムラインへ移行

## TIMELINE STAGE（横スクロール・可変幅）

タイムラインの全体像：

```
[時代区分ラベル]─────────────────────────────────────────────────
│
○ [ERA 01 イベントカード]
│
│                    ○ [ERA 02 イベントカード]
│                    │
│                               ○ [ERA 03 イベントカード]
│
───────────────[グロウライン: スクロールに応じて光が流れる]────────
```

**グロウライン：**
- 横軸に走る1本の光の線。スクロール位置に応じて「現在地」が発光
- CSSアニメーション（transform: scaleX()）でスクロール量に連動

---

## ERA（各時代カード・各100vw）

各ERAは同じ構造。時代の深刻さ・雰囲気に応じて背景色トーンを変化させる。

```
┌─────────────────────────────────────────┐
│                                         │
│  [時代区分ラベル]                        │
│  例：「第一紀元」「創世の時代」「200年前」│
│  ※ 作品固有の時代単位を使用             │
│                                         │
│  ┌────────────────────────────────────┐ │
│  │                                    │ │
│  │  [PLACEHOLDER: 時代のビジュアル]   │ │
│  │  コンセプトアート・地図・紋章等    │ │
│  │  ※ なし時：時代カラーの幾何学背景 │ │
│  │                                    │ │
│  └────────────────────────────────────┘ │
│                                         │
│  [PLACEHOLDER: 時代タイトル]             │
│  例：「神々の黄昏」                      │
│                                         │
│  [PLACEHOLDER: 時代説明テキスト 100〜200字]│
│  ※ スクロール位置到達でフェードイン     │
│                                         │
│  ┌──────────┐  ┌──────────┐             │
│  │  イベント①  │  │  イベント②  │           │
│  │ [アイコン] │  │ [アイコン] │           │
│  │ [タイトル] │  │ [タイトル] │           │
│  └──────────┘  └──────────┘             │
│                                         │
│  [ホバー時：詳細パネルがスライドで展開]  │
│                                         │
└─────────────────────────────────────────┘
```

**ERA構成例（5ERA）：**

| ERA | テーマ例 | 背景トーン |
|---|---|---|
| 01 | 創世・神話時代 | 最も深いネイビー（#050810） |
| 02 | 建国・拡張期 | ネイビー（#0a0e1a） |
| 03 | 動乱・戦争期 | 暗めのスレート（#0d1520） |
| 04 | 再建・現代 | 中間トーン（#111827） |
| 05 | 現在・物語の舞台 | やや明るいスレート（#151e2d） |

**イベントカード（ホバー展開仕様）：**
- 平常時：タイトル＋アイコンのみ表示
- ホバー・タッチ時：詳細テキスト（50〜100字）・関連キャラ名・関連地名がスライドで展開
- クリック時：背景が該当ERAカラーに切り替わり「その時代に入る」演出

## 拡張ロードマップ

1. ERA数は3〜8で調整可。設定資料が揃うにつれ追加
2. 各ERAにBGMを割り当てると、時代移動ごとに音楽が変化する（深い没入）
3. イベントカードにキャラクターポートレートを追加すると図鑑機能を兼ねる（Pattern G との融合）
4. 情報解禁が進んだ段階：CTA→Pattern F（Cinematic LP）へのリンクを設置
5. シリーズ展開時：ERA末尾に「第2部へ」のリンクでシームレスに続編タイムラインへ接続
