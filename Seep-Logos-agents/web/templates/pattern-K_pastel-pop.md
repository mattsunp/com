# ワイヤーフレーム：Pattern K「Pastel Pop型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：全年齢向け・ファミリー・女性向け作品。明るさ・かわいさ・親しみやすさが訴求軸
- **ページ数**：1ページ（縦スクロール）
- **必要最低素材**：ロゴ（丸みあり推奨）、キャラクタービジュアル、明るいブランドカラー
- **推奨スタイル**：パステルカラー・丸みUI・デフォルメキャラ・白背景基調・陽性トーン

## パステルポップカラーガイド

| 要素 | 推奨方針 |
|---|---|
| 背景 | 白（#FFFFFF）または超淡いパステル（明度90以上） |
| メインカラー | 作品のブランドカラー（明るめに設定） |
| アクセント | 補色または対比色を1〜2色（彩度高め） |
| フォント | 丸ゴシック・ポップ体（見出し）＋読みやすいゴシック（本文） |
| 装飾 | 星・ハート・泡・花・リボンなど作品テーマに応じて |
| ボタン | 角丸（border-radius大きめ）・パステル＋白テキスト |

---

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| HEROビジュアル | 明るいキャラクター集合絵 | パステルグラデーション＋装飾 |
| タイトルロゴ | 丸みのあるデザインロゴ | 丸ゴシックタイトルテキスト |
| キャラクター | デフォルメ or 立ち絵 | カラーブロック＋名前 |
| ストーリービジュアル | 冒険・仲間感のあるイラスト | カラーブロック |
| あらすじ | 200字以内・やさしい語り口 | 必須 |
| 特徴ビジュアル/アイコン | 各特徴を表すイラスト | カラーアイコン |

---

## 拡張ロードマップ

1. グッズ・コラボ展開：FEATUREセクション下にSHOP・COLLABタイルを追加
2. ゲームアプリの場合：「ストアでダウンロード」ボタン（App Store・Google Play）をHEROに追加
3. キャラ追加：CHARACTERSグリッドを拡張
4. 長期運用：Pattern Lへ移行しニュース・イベント情報を前面化
