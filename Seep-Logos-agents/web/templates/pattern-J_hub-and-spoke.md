# ワイヤーフレーム：Pattern J「Hub & Spoke型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：情報量が揃った大型タイトルのフルサイト。トップがハブとして全コンテンツへの入口になる
- **ページ数**：トップ（ハブ）＋各コンテンツページ（スポーク）のマルチページ構成
- **必要最低素材**：ロゴ、キービジュアル、各セクションのコンテンツ（ある程度揃っていることが前提）
- **推奨スタイル**：充実したナビゲーション、大型タイル、情報整理優先

---

## ページ構成（トップ/ハブ）

```
┌─────────────────────────────────────────┐
│  [NAV]（固定・多項目）                  │
├─────────────────────────────────────────┤
│  [SECTION 1] HERO                       │
├─────────────────────────────────────────┤
│  [SECTION 2] CONTENT TILES（メイン）     │
├─────────────────────────────────────────┤
│  [SECTION 3] PICKUP / LATEST            │
├─────────────────────────────────────────┤
│  [SECTION 4] NEWS                       │
├─────────────────────────────────────────┤
│  [SECTION 5] SNS / CTA                  │
├─────────────────────────────────────────┤
│  [FOOTER]（充実版）                     │
└─────────────────────────────────────────┘
```

---

## NAV（固定・多項目）

```
┌─────────────────────────────────────────┐
│ [ロゴ] STORY CHARACTERS SYSTEM MEDIA   │
│        NEWS  GOODS  SPECIAL  [購入する▶]│
└─────────────────────────────────────────┘
```

**要素メモ：**
- 項目数が多い場合は2段またはドロップダウン対応
- スマホ：ハンバーガーメニューに切り替え
- 「購入する」はCTAとして右端に強調配置

```
┌─────────────────────────────────────────┐

│  │                       │  │          ││
│  │  [PLACEHOLDER:        │  │[PLACE-   ││
│  │   世界観ビジュアル]   │  │HOLDER:   ││
│  │                       │  │キャラ集合 ││
│  │  物語の概要を見る →   │  │ビジュアル]││
│  │                       │  │          ││
│  └───────────────────────┘  │登場人物 →││
│                              └──────────┘│
│                                         │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐│
│  │          │  │          │  │          ││
│  │[TILE C:中]│  │[TILE D:中]│  │[TILE E:中]││
│  │SYSTEM    │  │MEDIA     │  │GOODS     ││
│  │          │  │          │  │          ││
│  │[PLACE-   │  │[PLACE-   │  │[PLACE-   ││
│  │HOLDER:   │  │HOLDER:   │  │HOLDER:   ││
│  │システム  │  │スクショ/ │  │グッズ    ││

│  │          │  │          │  │          ││
│  │ゲームを  │  │メディアを │  │グッズを  ││
│  │知る →   │  │見る →    │  │見る →   ││
│  └──────────┘  └──────────┘  └──────────┘│
│                                         │
│  ┌──────────────────────────────────────┐│
│  │  [TILE F: 横長]                      ││
│  │  SPECIAL / COLLABORATION             ││
│  │  [PLACEHOLDER: スペシャルビジュアル]  ││

│                                         │
└─────────────────────────────────────────┘
```

**タイル設計のルール：**
| タイル | 推奨サイズ | 必須度 |
|---|---|---|
| STORY | 大（2:1比率） | 必須 |
| CHARACTERS | 中 | 必須 |
| SYSTEM | 中 | ゲームのみ |
| MEDIA | 中 | トレーラー/スクショあれば |
| GOODS | 中 | グッズ展開あれば |
| SPECIAL | 横長 | コラボ/特典あれば |

## スポークページ構成（各コンテンツページ）

各タイルからリンクする独立ページ。共通のNAV・FOOTERを持つ。

| ページ | 主なコンテンツ |
|---|---|
| STORY | あらすじ・世界観・用語集 |
| CHARACTERS | Pattern Gのキャラクター図鑑を流用 |
| SYSTEM | ゲームシステム説明・インフォグラフィック |
| MEDIA | PV・スクリーンショットギャラリー |
| GOODS | グッズ一覧・通販リンク |
| SPECIAL | コラボ・キャンペーン・特典情報 |
| NEWS | ニュース一覧・カテゴリフィルター |

---

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| HEROビジュアル | キービジュアル | ブランドカラーグラデーション |
| TILE各背景 | 各コンテンツを象徴するビジュアル | カラーブロック＋テキスト |
| PICKUP | 最新注目コンテンツ | 省略可 |
| NEWS | 各記事 | 「準備中」行 |

---

## 拡張ロードマップ

1. 初期：タイルは3〜4個から始め、コンテンツが揃い次第追加
2. DLC・続編：SPECIALタイルを追加・更新
3. メディアミックス展開：各メディア（アニメ・コミック・グッズ）をタイルに追加
4. 最終形：多数のタイルが並ぶPattern Lの複合ポータル型へ進化
