# ワイヤーフレーム：Pattern A「Teaser型」

作成日：2026-03-19
担当エージェント：デジタル・技術部 / web-director
実装担当エージェント：不明（ログ導入前）

## 概要
- **用途**：情報解禁前のティザーサイト。最小限の素材で最大のインパクト
- **ページ数**：1ページ（シングルスクロール）
- **必要最低素材**：ロゴ（またはゲームタイトルテキスト）、キャッチコピー、ブランドカラー
- **推奨スタイル**：Dark UI、フルスクリーン、タイポグラフィ中心

---

```
│                                         │
├─────────────────────────────────────────┤
│  [FOOTER]                               │
└─────────────────────────────────────────┘
```

## SECTION 1：HERO（フルスクリーン・100vh）

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│         [PLACEHOLDER: 背景ビジュアル]    │
│         ※ 素材なし時はブランドカラーの   │
│           グラデーションで代替可         │
│                                         │
│    ┌───────────────────────────────┐    │
│    │  [PLACEHOLDER: ゲームロゴ]    │    │
│    │  または タイトルテキスト       │    │
│    └───────────────────────────────┘    │
│                                         │
│    [PLACEHOLDER: キャッチコピー 1行]     │
│                                         │
│    ── COMING SOON ──                    │
│                                         │
│    ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ │
│    │ 00日 │ │ 00時 │ │ 00分 │ │ 00秒 │ │
│    └──────┘ └──────┘ └──────┘ └──────┘ │
│    ※ 解禁日未定の場合はカウントダウン非表示│
│                                         │
│    ▼ スクロール                          │
│                                         │
└─────────────────────────────────────────┘
```

**要素メモ：**
- 背景：フルスクリーン。素材なし時はグラデーション（例：深紺→黒）
- ロゴ：SVG推奨。なければゲームタイトルをセリフ体テキストで代替
- カウントダウン：発売日/解禁日確定後に実装。未定時は非表示
- アニメーション：ロゴのフェードイン、パーティクル（軽量）

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| 背景ビジュアル | キーアート・コンセプトアート | ブランドカラーのグラデーション |
| ゲームロゴ | SVGロゴデータ | タイトルテキスト（セリフ体） |
| キャッチコピー | 1行のゲームコンセプト文 | 「詳細は近日公開」等 |
| カウントダウン | 解禁・発売日 | 非表示（coming soonのみ） |
| SNSアイコン | 各SNS公式URL | 開設済みのみ表示 |

## 拡張ロードマップ（素材が揃い次第追加）

1. キーアート入手 → 背景ビジュアルに差し替え
2. PV完成 → HEROにYouTube埋め込みボタン追加
3. キャラクター確定 → Pattern Bへ移行 or セクション追加
4. 発売日確定 → カウントダウン実装
