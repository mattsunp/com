# ワイヤーフレーム：Pattern O「Card Carousel型」

作成日：2026-03-31
担当エージェント：デジタル・技術部 / web-director
UIレビュー：アートディレクション部 / art-director、game-ui-designer

## 概要
- **用途**：更新頻度の高いコンテンツ運用サイト。ニュース・新キャラ追加・イベント情報・グッズ・コラボなど、多品種・高頻度の更新をカード形式で集約して見せる
- **ページ数**：1ページ（メインLP）
- **必要最低素材**：ロゴ、カードコンテンツ（タイトル・日付・カテゴリ・本文50〜100字）、ブランドカラー
- **推奨スタイル**：ダーク基調＋アクセントカラー（カテゴリ別に色分け）、カード浮遊感、滑らかなカルーセルアニメーション

---

## 差別化ポイント（対 A〜M）

| 軸 | 本パターンの立場 |
|---|---|
| vs Pattern L（News Feed） | Lは縦スクロール型のリスト閲覧（情報アクセス優先）。Oはカルーセル横回遊（発見・探索優先）。Lが「読む」体験ならOは「めくる」体験 |
| vs Pattern D（Bento-reveal） | Dは静的グリッド配置（固定枠）。Oは動的に追加・切り替え可能なカードストリーム。運営更新への対応が核心的差異 |
| vs Pattern G（Character Encyclopedia） | Gはキャラ図鑑（縦列閲覧・1キャラ深掘り）。Oは多品種コンテンツの横断的回遊 |
| vs Pattern J（Hub-and-Spoke） | Jはリンク集（分岐起点）。Oはコンテンツそのものをその場で消費させる設計 |
| 独自性 | カテゴリタブ＋カルーセル＋カードホバー展開の三層インタラクション構造は A〜M に存在しない |

**要素メモ：**

- ナビ項目は最小限。情報をカルーセルHUBに集中させる設計

## SECTION 1：HERO（コンパクト・50vh）

```
┌─────────────────────────────────────────┐
│                                         │
│  [PLACEHOLDER: 背景ビジュアル]          │
│  ※ なし時：ブランドカラーのグラデーション│
│                                         │
│  [PLACEHOLDER: タイトルロゴ]            │
│                                         │
│  [PLACEHOLDER: キャッチコピー 1〜2行]   │
│  例：「すべての最新情報が、ここにある。」 │
│                                         │
│  ↓ カードを探す                         │
│  （下矢印アニメーション・カルーセルへ誘導）│
│                                         │
└─────────────────────────────────────────┘
```

**アニメーション：**
- ロゴ・キャッチコピーがフェードイン
- 下矢印が上下バウンスアニメーション
- HEROはコンパクトに収め、CAROUSEL HUBへの視線誘導を最優先

## SECTION 2：CAROUSEL HUB（メイン）

カテゴリタブとカルーセルの三層インタラクション構造：

### 第1層：カテゴリタブ

```
┌─────────────────────────────────────────┐
│                                         │
│  [すべて]  [ニュース]  [キャラクター]   │
│  [イベント]  [グッズ]  [コラボ]         │
│                                         │
│  ※ アクティブタブはアクセントカラー表示 │
│  ※ カテゴリ数は運用に応じて増減可       │
└─────────────────────────────────────────┘
```

**動作：**
- タブ切り替えでカルーセルのカードセットがアニメーションで入れ替わる（フェード＋スライド）
- 各カテゴリにカラーアクセントを割り当て（視覚的識別）
- 「すべて」では全カテゴリのカードが時系列逆順で混在表示

### 第2層：カルーセル

```
┌─────────────────────────────────────────┐
│                                         │
│  ◁  ┌────────┐  ┌────────┐  ┌────────┐  ▷ │
│     │        │  │        │  │        │    │
│     │[カード]│  │[カード]│  │[カード]│    │
│     │        │  │        │  │        │    │
│     └────────┘  └────────┘  └────────┘    │
│                                         │
│         ● ○ ○ ○ ○  （ページネーション）  │
│                                         │
└─────────────────────────────────────────┘
```

**挙動：**

| 操作 | 挙動 |
|---|---|
| ← → ボタン | 1枚ずつスライド（慣性あり） |
| マウスドラッグ | 横スクロール |
| タッチスワイプ | 横スクロール（慣性あり） |
| キーボード（←→） | 1枚ずつ移動 |
| 自動スライド | 5秒間操作なしで自動進行（ホバー時一時停止） |

- 表示枚数：デスクトップ3枚・タブレット2枚・モバイル1枚（レスポンシブ）
- ループ再生：最後のカードの次は最初のカードへ循環

### 第3層：カード

```
┌─────────────────────────────────────┐
│                                     │
│  [カテゴリバッジ]  2026.XX.XX       │
│                                     │
│  ┌─────────────────────────────┐   │
│  │                             │   │
│  │  [PLACEHOLDER: カード画像]  │   │
│  │  ※ なし時：カテゴリカラー  │   │
│  │       + タイトル頭文字      │   │
│  │                             │   │
│  └─────────────────────────────┘   │
│                                     │
│  [PLACEHOLDER: カードタイトル]      │
│  例：「第3キャラクター公開」         │
│                                     │
│  [PLACEHOLDER: 本文50〜100字]       │
│  ※ 平常時は2行まで表示             │
│                                     │
│  [ → 詳細を見る ]                  │
│                                     │
└─────────────────────────────────────┘
```

**カードインタラクション：**
- 平常時：タイトル＋画像＋2行テキスト＋CTAボタン
- ホバー時：カードが浮き上がる（translateY -6px＋box-shadow強化）
- ホバー時：本文が全文展開（最大100字）
- フォーカス時（キーボード）：ホバーと同等のスタイル（アクセシビリティ対応）

**カードに必要な最小データ：**
```
- id: card-001
  category: "ニュース"               # カテゴリ（タブ連動）
  date: "2026.03.31"
  title: "カードタイトル（必須）"
  body: "本文テキスト50〜100字（必須）"
  image: ""                          # 省略可（なし時はカテゴリカラーで代替）
  link: "#"                          # 省略可
```

## SECTION 3：SPOTLIGHT（注目コンテンツ・60vh）

カルーセルとは別枠で、運営が手動で「押したい」コンテンツを大きく表示する。

```
┌─────────────────────────────────────────┐
│                                         │
│  ── PICKUP ──                           │
│                                         │
│  ┌─────────────────────┐  ┌──────────┐  │
│  │                     │  │          │  │
│  │ [PLACEHOLDER:       │  │[タイトル]│  │
│  │  メインビジュアル   │  │          │  │
│  │  ※ なし時：グラデ] │  │[本文     │  │
│  │                     │  │ 100〜200字│  │
│  │                     │  │ ]        │  │
│  │                     │  │          │  │
│  │                     │  │[→ 詳細] │  │
│  └─────────────────────┘  └──────────┘  │
│                                         │
│  ← →（複数SPOTLIGHTがある場合）         │
│                                         │
└─────────────────────────────────────────┘
```

**運用メモ：**
- SPOTLIGHTは1〜3件まで。多すぎるとカルーセルHUBと重複する
- 「今週のイチ押し」「期間限定イベント」「新キャラ大公開」など、時限性の高い情報に使う
- 表示期間が過ぎたらカルーセルHUBのカードへ降格運用を想定

## 運用フロー（カード追加の想定）

```
1. カードデータ（title/body/category/date）を配列に追記
2. 保存するだけ → カルーセルに自動挿入（最新日付順）
3. カテゴリを指定していれば自動でタブ振り分け
4. SPOTLIGHTは別途 spotlight 配列で管理（手動選定）
```

---

## 拡張ロードマップ

1. カード数が増えたら「もっと見る」ボタンで追加読み込み（ページネーション不要）
2. カード検索機能：タイトル・カテゴリで絞り込み
3. お気に入り機能：LocalStorageでカード保存
4. 新着バッジ：追加から7日間は「NEW」バッジを自動付与
5. Pattern G（キャラ図鑑）との連携：キャラカテゴリのカードクリックで図鑑ページへ遷移
6. Pattern L（News Feed）との連携：ニュースカテゴリのカードが蓄積したらL型のアーカイブページへリンク
