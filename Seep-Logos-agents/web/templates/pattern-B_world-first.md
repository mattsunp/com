# ワイヤーフレーム：Pattern B「World-first型」

作成日：2026-03-19
担当エージェント：デジタル・技術部 / web-director
実装担当エージェント：不明（ログ導入前）

## 概要
- **用途**：世界観・雰囲気を前面に出す。物語のある作品向き（RPG、ADVなど）
- **ページ数**：1ページ（縦スクロール・複数セクション）
- **必要最低素材**：ロゴ、キャッチコピー、世界観テキスト（数百字）、ブランドカラー
- **推奨スタイル**：Full-screen Scroll、Dark UI、余白多め

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| HEROビジュアル | キーアート | ブランドカラー＋テクスチャ |
| ゲームロゴ | SVGロゴ | タイトルテキスト |
| キャッチコピー | 2行のコンセプト文 | 必須（テキストのみでOK） |
| コンセプトアート | 世界観イラスト | カラーブロック＋引用テキスト |
| 世界観テキスト | 300〜500字 | 必須（最低限の文章が必要） |
| キャラクター立ち絵 | キャラクタービジュアル | シルエット＋「？」演出 |
| NEWSコンテンツ | 記事 | 「近日公開予定」行 |

## 拡張ロードマップ

1. キャラクタービジュアル入手 → シルエットを差し替え
2. PV完成 → HEROにPV再生ボタン追加
3. スクリーンショット入手 → WORLD下にGALLERYセクション追加
4. 発売日確定 → NAVに「購入する」ボタン追加、CTAをストア誘導に変更
