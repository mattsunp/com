# ワイヤーフレーム：Pattern F「Cinematic LP型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：発売直前・リリース告知。動画（トレーラー）を主役に購入・予約を促す
- **ページ数**：1ページ（縦スクロール）
- **必要最低素材**：トレーラー動画（YouTube/Vimeo埋め込み可）、ロゴ、購入先URL
- **推奨スタイル**：動画BG・シネマティック・CTA優先・ダーク〜ミッドトーン

## NAV（固定・最小限）

```
┌─────────────────────────────────────────┐
│ [ロゴ]          TRAILER  GALLERY  購入▶ │
└─────────────────────────────────────────┘
```

**要素メモ：**
- 項目は4つ以内に絞る
- 「購入▶」は発売前は「予約する」、発売後は「購入する」に切り替え

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| VIDEO HERO背景 | ループ動画 | キーアート静止画 |
| タイトルロゴ | SVGロゴ | タイトルテキスト |
| キャッチコピー | 1〜2行 | 必須 |
| トレーラー動画 | YouTube/Vimeo URL | セクション非表示 |
| FEATURES ビジュアル | 各特徴を表す画像/アイコン | 大きな数字・テキストで代替 |
| スクリーンショット | ゲーム/アニメ場面カット | セクション非表示（後から追加） |
| 購入先URL | 各ストアのURL | 発売前は「予約」に変更 |

---

## 拡張ロードマップ

1. 発売前：HERO CTAを「予約する」、PURCHASE CTAも「予約」表記
2. 発売日：全CTAを「購入する」に一斉切り替え
3. DLC・アップデート時：FEATURESセクションを更新
4. 長期運用移行時：Pattern Lへ切り替え、ニュースフィードを前面に
