# ワイヤーフレーム：Pattern L「News Feed型」

作成日：2026-03-30
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：放映中TVアニメ・長期運用ライブサービスゲーム。既存ファンが週次/定期的に訪れる運用サイト
- **ページ数**：1ページ（トップ）＋各コンテンツページ（ニュース一覧・キャラ等）
- **必要最低素材**：ロゴ、更新情報（NEWS）、放映/配信スケジュール、キービジュアル
- **推奨スタイル**：ニュース中心・情報アクセス最優先・定期更新を前提とした運用しやすい設計

---

## SECTION 1：HERO（コンパクト・60vh）

```
┌─────────────────────────────────────────┐
│                                         │
│  ┌──────────────────┐  ┌─────────────┐  │
│  │                  │  │             │  │
│  │[PLACEHOLDER:     │  │[PLACEHOLDER:│  │
│  │ 最新キービジュアル│  │ タイトルロゴ│  │
│  │ （クール更新時    │  │             │  │
│  │  に差し替え）]   │  │[キャッチコピ│  │
│  │                  │  │ー 1〜2行]  │  │
│  │                  │  │             │  │
│  │                  │  │ 放送中      │  │
│  │                  │  │ [放送局]    │  │
│  │                  │  │ [曜日・時間]│  │
│  └──────────────────┘  └─────────────┘  │
```

**要素メモ：**
- HEROはコンパクトに。NEWSフィードを早く見せる
- 「放送中」バッジで運用中感を演出
- キービジュアルは最新クール/シーズンのものに定期更新

## 運用カレンダー（週次更新の想定）

| タイミング | 更新内容 |
|---|---|
| 毎週（放映日前日） | 次回あらすじ・サムネイルを更新 |
| 毎週（放映日当日） | 「放映済み」バッジ・話数更新 |
| 随時 | NEWS記事追加、グッズ情報、コラボ情報 |
| 季節 | キービジュアルのクール更新 |
| イベント | PICKUPバナーの差し替え |

---

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| HEROキービジュアル | 最新クールKV | ブランドカラーグラデーション |
| NEWS記事 | 各記事テキスト・サムネイル | カテゴリカラーブロック |
| 放送スケジュール | 各局・配信サービスの放送日時 | 必須（これが主役） |
| 次回あらすじ | 50〜100字 | 「近日公開」 |
| ピックアップバナー | 注目コンテンツの画像 | カラーブロック＋テキスト |
