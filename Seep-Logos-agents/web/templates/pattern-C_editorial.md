# ワイヤーフレーム：Pattern C「Editorial型」

作成日：2026-03-19
担当エージェント：デジタル・技術部 / web-director

## 概要
- **用途**：タイポグラフィと断片的な情報発信を主役にしたティザー。インディ・アート系ゲーム向き
- **ページ数**：1ページ（縦スクロール）
- **必要最低素材**：タイトル、キャッチコピー、世界観テキスト断片（短文×複数）、ブランドカラー
- **推奨スタイル**：Editorial、Minimalist、タイポグラフィ主役

---

## SECTION 1：HERO（タイポグラフィ主役・100vh）

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│  ██████████████████████████████████     │
│  █                                      │
│  █  [PLACEHOLDER: ゲームタイトル]        │
│  █  大きなセリフ体 or サンセリフ体       │
│  █  画面幅いっぱいに配置                │
│  █                                      │
│  ██████████████████████████████████     │
│                                         │
│                          [PLACEHOLDER:  │
│                          キャッチコピー  │
│                          2〜3行         │
│                          右寄せ]        │
│                                         │
│  ── 2026  [ジャンル]  [対応機種] ──     │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

**要素メモ：**
- タイトルテキストは画面幅いっぱい（fluid typography）
- 背景：白・黒・またはブランドカラー単色
- ロゴ画像がなくてもテキストのみで成立する唯一の型

## SECTION 3：FRAGMENTS（世界観断片・120vh）

```
┌─────────────────────────────────────────┐
│                                         │
│  ┌─────────────┐                        │
│  │[PLACEHOLDER:│  [世界観テキスト断片①] │
│  │ 小さめの    │  短文・詩的・謎めいた   │
│  │ ビジュアル  │  表現でOK              │
│  │ or カラー  │                        │
│  │ ブロック]  │                        │
│  └─────────────┘                        │
│                                         │
│              ─────────────────────────  │
│              [世界観テキスト断片②]      │
│              ─────────────────────────  │
│                          ┌────────────┐ │
│                          │[PLACEHOLDER│ │
│                          │ ビジュアル │ │
│                          │ or カラー  │ │
│                          │ ブロック]  │ │
│                          └────────────┘ │
│                                         │
│  [世界観テキスト断片③]                  │
│                                         │
└─────────────────────────────────────────┘
```

**要素メモ：**
- テキスト断片は世界観の「断片」でよい。完全な説明は不要
- 例：「その大陸に、神はいなかった。」「3000年後の記憶」等
- ビジュアルブロックはコンセプトスケッチ・テクスチャ・カラーいずれでもOK
- 左右交互に配置してリズムを出す

## プレースホルダー一覧

| 箇所 | 内容 | 素材なし時の代替 |
|---|---|---|
| ゲームタイトル | テキストで代替可（この型の強み） | **代替不要・テキストが主役** |
| キャッチコピー | 2〜3行のコンセプト文 | 必須 |
| KEY VISUAL | キービジュアル | カラーブロック＋引用テキスト |
| 世界観断片テキスト | 短文×3 | 必須（数行でOK） |
| FRAGMENTSビジュアル | スケッチ・テクスチャ等 | ブランドカラーブロックで代替可 |

## 拡張ロードマップ

1. キービジュアル入手 → SECTION 2に差し替え
2. キャラクター確定 → FRAGMENTSにキャラクター紹介を追加
3. PV完成 → HEROにPV再生ボタン追加
4. 情報解禁 → SECTION 3をWORLD・CHARACTERSページに分割してマルチページ化
