# ワイヤーフレーム：Pattern D「Bento Reveal型」

作成日：2026-03-19
担当エージェント：デジタル・技術部 / web-director
実装担当エージェント：不明（ログ導入前）

## 概要
- **用途**：情報を段階的に解放していく謎解き型。情報公開をイベント化できる
- **ページ数**：1ページ（ファーストビュー＋ベントグリッド）
- **必要最低素材**：ロゴ、キャッチコピー、ブランドカラー（ブロックが空でも「謎」として機能する）
- **推奨スタイル**：Bento Grid、Dark UI、モジュール型

## ブロック状態の定義

| 状態 | 見た目 | 用途 |
|---|---|---|
| **公開済み** | コンテンツ表示 | 情報解禁済みのブロック |
| **近日公開** | 🔒 ＋ 解禁日表示 | 解禁日が決まっているブロック |
| **未公開** | 🔒 ＋ 「？」のみ | 解禁日未定のブロック |

**演出メモ：**
- 近日公開ブロックはカウントダウンタイマーを埋め込み可
- 解禁時にアニメーションでブロックが「開く」演出を加えると情報公開がイベント化する
- SNSでの「○○が解禁されました！」告知と連動させると効果的

## ブロック構成の初期状態（スターター）

| ブロック | 初期状態 | 最終形 |
|---|---|---|
| A: WORLD | 世界観テキスト（必須） | 世界観ビジュアル＋テキスト |
| B: CHARACTER 01 | シルエット＋近日公開 | キャラクタービジュアル＋プロフィール |
| C: CHARACTER 02 | 🔒 未公開 | キャラクタービジュアル＋プロフィール |
| D: STORY | 短文 or 🔒 | ストーリーテキスト＋ビジュアル |
| E: SYSTEM | 🔒 未公開 | ゲームシステム説明 |
| F: NEWS | 空リスト or「近日公開」 | ニュース記事リスト |
| G: MEDIA | 🔒 未公開 | スクリーンショット・PVギャラリー |
| H: PRE-REG | 常時表示 | → 発売後はストア誘導に変更 |

## プレースホルダー一覧

## 拡張ロードマップ

1. 情報解禁のたびにブロックを「公開済み」に更新
2. キャラクター追加ごとにブロック追加（グリッドは柔軟に拡張可）
3. 発売日確定 → ブロックHをストア購入誘導に変更
4. 発売後 → 全ブロック公開状態＋レビュー・メディア掲載ブロック追加
