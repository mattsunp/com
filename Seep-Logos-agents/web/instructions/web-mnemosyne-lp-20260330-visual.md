## 概要

- **目的**: Steamおよびitch.ioでの短編有料DL販売に向けた事前認知獲得LP
  ゲームの世界観と感情体験を先行して伝え、リリース時のウィッシュリスト登録・購入へ誘導する。
  Seep Logosのスタジオクリエイティブ水準を示す名刺代わりの作品としても機能させる。
- **使用媒体**: Web LP（モバイルファースト。SNS流入を主要経路と想定）
- **状態**: 素材ゼロ状態からの着手。キービジュアル・ロゴ・キャラクターイラスト未確定。
  テキスト・幾何学・タイポグラフィのみで世界観を構築する。
- **言語**: 日本語のみ（初期フェーズ）

## デザインコンセプト

**「忘却の静寂」**

深淵な暗黒の中に、崩壊しかけた記憶の断片が漂う。
デジタルとアナログの境界が溶けた2040年代の荒廃した静けさ。
美しさではなく、問いの重さを体で感じさせるビジュアル言語。

訪問者は説明を受けるのではなく、問いの入口に立つ。
「忘れることは、罪か。」という核心の問いをLPそのもので体験する設計。

## カラーシステム

### 使用カラー

| 役割 | カラーコード | 用途 |
|---|---|---|
| メインBG | `#0A0A0F` | 全セクションの基本背景（深宇宙色） |
| メインテキスト | `#F0EDE8` | 本文・コピー・見出し（くすんだオフホワイト） |
| サブ1 | `#4A7C9E` | ラインアクセント・システム用語断片（錆びた青） |
| サブ2 | `#8B8680` | 副テキスト・注記・フェードした要素（消えかけグレー） |
| アクセント | `#C8A882` | 核心コピーへの最小限の強調・ボーダー（褪せたアンバー） |
| カード背景 | `#0F0F14` | Sectionカード類の浮きBG |

### 禁止カラー

- `#FFFFFF`（純白）——コントラストが鋭すぎて没入を破壊する
- `#000000`（純黒）——同上。必ず `#0A0A0F` を使う
- 彩度の高い暖色系（赤・オレンジ・黄）——感情的刺激が世界観の静けさを破壊する
- 青紫系の明るいグラデーション——tech感・未来感が出すぎてしまう

### カラー使用原則

- アクセントカラー（`#C8A882`）は1ページあたり最大3箇所まで。乱用禁止。
- グラデーション背景は全面禁止。ベタ塗りのみ使用。
- 透明度（opacity）の調整でのみトーン変化を表現する。

## タイポグラフィ仕様

### フォントファミリー

| 用途 | フォント | ウエイト |
|---|---|---|
| 日本語メインコピー・本文 | Noto Serif JP | 100〜300（細ウエイトのみ） |
| 英字見出し・タイトル | Cormorant Garamond | Light（300）または Regular（400）細ウエイト |
| 英字サブコピー | Playfair Display | 細ウエイト（代替案） |
| システムUI・数値・コード断片 | JetBrains Mono | Regular（MET用語・座標・コード断片のみに限定） |

### タイポグラフィルール

- 日本語ゴシック系フォント禁止（ページ全体を通じて）
- ヒーローコピーは大胆に大きく（vw単位推奨。デスクトップで72px〜96px相当）
- 本文は読みやすい最小サイズで抑制（モバイル: 15px〜16px、デスクトップ: 16px〜17px）
- 行間（line-height）: 1.9〜2.2（余白が世界観を語る）
- 文字カーニング（letter-spacing）: 緩め。日本語 0.05em〜0.1em、英字 0.08em〜0.15em
- 詰まった文字は禁止。窮屈さは没入を妨げる。

### テキストサイズ階層

| レベル | 用途 | サイズ目安（デスクトップ） |
|---|---|---|
| Hero Copy | 核心コピー「忘れることは、罪か。」 | 64px〜96px |
| Title | MNEMOSYNE（英字タイトル） | 24px〜32px |
| Section Heading | セクション見出し | 28px〜36px |
| Sub Copy | 副コピー・状況説明 | 16px〜20px |
| Body | 本文 | 15px〜16px |
| Micro / System | MET用語・座標・注記 | 11px〜13px（モノスペース） |

## モーション方針

### 基本原則

「世界観の静けさを壊さない範囲で最小限のモーション」

- アニメーションはすべてease-inまたはease-out（線形・バウンスは禁止）
- duration は長めに取る（0.6s〜1.2s）。素早い動きは世界観と合わない
- スクロールのたびに要素がポップインする演出は全面禁止

### 許可するモーション

| 要素 | モーション | 仕様 |
|---|---|---|
| スクロール誘導矢印 | 上下フェード（呼吸感） | opacity 0.3 → 1 → 0.3、duration 2s、loop |
| テキスト表示 | フェードイン（opacity 0 → 1） | duration 0.8s〜1.0s、スクロールで一度だけ発火 |
| CTAボタン hover | ボーダーが淡く光る | box-shadow or border-color の opacity 変化。duration 0.4s |
| ノイズ/スキャンライン | 静止または極低速フロート | 動く場合は30s以上のduration。気づかない程度に |

### 禁止するモーション

- スクロールパララックスによる背景画像移動
- 要素のスライドイン（横から飛び込む演出）
- スケールアップ・バウンスアニメーション
- カウントアップ・カウントダウン数値アニメーション
- 点滅（blink）——フェードで代替する

## セクション別ビジュアル仕様

### Section 1 — HERO（フルスクリーン）

**目的**: タイトルと核心の問いで訪問者を世界の入口に立たせる

**レイアウト**
- 背景: `#0A0A0F` フルスクリーン（100vh）
- 極低透明度（opacity: 0.04〜0.06）のスキャンラインまたはノイズテクスチャを重ねる
  （CSSのpseudo-elementでSVGノイズパターンを実装。外部画像ファイル不要）
- コンテンツは画面中央〜やや上に縦積み配置（flex + column）

**テキスト配置（上から順）**
1. `忘れることは、罪か。`
   - Noto Serif JP Weight 200、Hero Copyサイズ（モバイル: 36px〜42px / デスクトップ: 72px〜88px）
   - 文字色: `#F0EDE8`
   - letter-spacing: 0.1em
2. `MNEMOSYNE`
   - Cormorant Garamond Light
   - コピーの下に間隔を置いて小さく配置（モバイル: 16px / デスクトップ: 22px）
   - 文字色: `#8B8680`（抑制した存在感）
   - letter-spacing: 0.3em
3. 副コピー（任意表示。存在を主張しすぎない）
   - 「他者の記憶を消す仕事を10年続けた——そして自分の消された記憶に気づいた」
   - Noto Serif JP Weight 100、body サイズ
   - margin-top を広くとる（コピーとの間に沈黙を作る）

**スクロール誘導**
- 極細ライン（1px、`#8B8680`）の下向き矢印（またはシンプルな縦ライン）
- フェードで呼吸させる（opacity変化、点滅禁止）
- 画面下端から 40px〜60px 上に絶対配置

**NG**
- ロゴ・アイコン・装飾的イメージ素材の配置（素材なし前提）
- 明るい色・グラデーション
- 自動再生される動画・GIF背景

---

### Section 2 — WORLD（世界観紹介）

**目的**: 2040年代の日本・記憶外在化技術（MET）という世界の輪郭を伝える

**レイアウト**
- 背景: `#0A0A0F`（継続）
- 左寄せのテキストブロックと右側に散在する数値・記号断片で非対称構成
  （デスクトップ: 2カラム非対称グリッド。左70% / 右30%）
  （モバイル: 1カラム縦積み。数値断片はテキストの後に配置）
- padding は大きめ（上下 80px〜120px、左右 24px〜48px）

**テキストブロック（左側）**
- `西暦2040年代。記憶は、整理できる時代になった。` — 導入コピー
- 世界観を示す短い断片テキストを2〜4段落で積み上げる
- 説明文ではなく、世界の空気を漂わせるトーン
- Noto Serif JP Weight 200、body サイズ、line-height 2.0〜2.2

**数値・記号断片（右側）**
- `MET v4.2`、`ACCESS: BLOCKED`、`LEAK DETECTED` 等
- JetBrains Mono、Microサイズ（12px〜13px）
- 文字色: `#4A7C9E`（錆びた青）
- 各断片の間に余白を設け、整列はしない（散在感を出す）
- ラインセパレーター: 1px未満の細さ（0.5px）、`#4A7C9E` 20%opacity

**NG**
- ドット・グラフ・インフォグラフィック的なビジュアル化
- Font Awesomeなどのアイコンフォント
- 情報密度が高すぎる詰め込みレイアウト

---

### Section 3 — STORY（物語・主人公紹介）

**目的**: 桐島セツという人物と、彼女の置かれた状況を伝える

**レイアウト**
- 背景: `#0A0A0F`（継続）
- 中央揃え（テキストのみ構成のため、中央に重心を置く）
- キャラクターイラスト未確定のため、シルエット的な暗色領域を右または左に配置
  （CSSのみで実装。`#0F0F14` の長方形または楕円形。overflow: hidden でエッジ処理）

**テキスト配置**
- 名前の上下に大きな余白を設け、人物の"重さ"を余白で表現する
  例:
  ```
  [margin: 60px〜80px]

  桐島 セツ

  [margin: 40px〜60px]

  記憶消去の技術者。37歳。
  10年間、他者の記憶を消してきた。
  ```
- `桐島 セツ` の文字: Noto Serif JP Weight 100、大きめ（28px〜36px）
- 職業・年齢・状況: Weight 200、bodyサイズ
- 物語の核心（失っているもの・恐れているもの）を問いかける形で記述

**暗色領域の扱い**
- 実体のない影として、テキストの背景に溶け込む程度に留める
- opacity: 0.4〜0.6。完全に塗りつぶさない
- 境界はぼかし（blur）または透明度グラデーション（CSS radial-gradient）

**NG**
- キャラクターイラストの代替として人物写真・ストック写真を使う
- 明確なシルエット図形（人型のアイコン等）——抽象的な暗色領域に留める

### Section 4 — SYSTEM（ゲームプレイ）

**目的**: 記憶の分類というゲームプレイの体験を言葉で先取りさせる

**レイアウト**
- 背景: `#0A0A0F`（継続）
- 3つの選択カードを横並び（デスクトップ）または縦積み（モバイル）で配置
- カード間の余白を十分に取る（gap: 24px〜40px）

**カード仕様（3枚共通）**
- 背景: `#0F0F14`（わずかに浮いたダーク）
- ボーダー: 1px solid `#8B8680`（30%〜40%opacity）
- padding: 32px〜48px
- box-shadow: なし（浮き上がるシャドウは禁止）
- border-radius: 0（または最大4px。丸みをつけない）

**カード内テキスト**
- カードタイトル（整理 / 保持 / 封鎖）: Noto Serif JP Weight 200、20px〜24px
- カード説明文: Noto Serif JP Weight 100、body サイズ
- アクセントライン: タイトル下に 1px の `#4A7C9E` ラインを短く引く（幅: 24px〜32px）
- アイコンを使う場合: 極細ラインの幾何学形のみ（SVGで実装。Font Awesomeは禁止）

**導入コピー（カードの上）**
- 「あなたは記憶の断片を受け取る。整理するか、保持するか、封鎖するか。」
- 「正解はない。しかし、選択には重さがある。」
- プレイ時間（90〜120分）とエンディングの存在をさりげなく、小さいテキストで示す

**NG**
- カードへのhover時の浮き上がりアニメーション（transform: translateY）
- カラーでの3カード色分け（統一感を保つ）
- 評価・スター・数値化されたゲームシステム説明

### Section 5 — CTA（ウィッシュリスト・メール登録）

**目的**: 感情的に高まった状態でアクションを促す

**レイアウト**
- 背景: `#0A0A0F`（HEROセクションと同系統の深黒に戻る）
- 中央揃え、縦積み構成
- padding: 上下 100px〜140px（十分な呼吸を持たせる）

**テキスト配置（上から順）**
1. `記憶は、あなたが選ぶ。`
   - Noto Serif JP Weight 200、大きめ（28px〜36px）
   - ボタン上方に静かに配置
   - 文字色: `#F0EDE8`
2. `MNEMOSYNE`（タイトル再掲）
   - 小さく・薄く（`#8B8680`、14px〜16px、letter-spacing: 0.3em）
   - コピーの下、ボタンの上
3. 主CTAボタン: 「Steamウィッシュリストに追加」
4. 副CTAリンク: 「リリース通知をメールで受け取る」（ボタンの下、テキストリンク形式）

**CTAボタン仕様**
- 塗りつぶしなし（ゴースト型）
- ボーダー: 1px solid `#C8A882`（アンバー）
- テキスト色: `#C8A882`
- padding: 14px 40px〜48px
- font: Cormorant Garamond Light または Noto Serif JP Weight 200
- hover: border-color と text-color の opacity が淡く増す（0.7 → 1.0）。box-shadow は淡い発光程度
- border-radius: 0（角丸なし）

**フッター**
- SNSアイコン: SVGの極細ラインアイコンのみ（Font Awesomeは禁止）
- 著作権表記: `© 2026 Seep Logos`、Microサイズ、`#8B8680`
- CTAボタンとフッターの間に十分な余白

**NG**
- 塗りつぶしボタン（bright fill）
- 「今すぐ買う」「期間限定」等の煽り表現
- カウントダウンタイマー
- ソーシャルプルーフ（フォロワー数・評価数）

## 全体共通禁止事項

以下は全セクションを通じて禁止する。

| 禁止要素 | 理由 |
|---|---|
| グラデーション背景 | ブランド感・明るさが出て世界観を破壊する |
| 浮き上がるカードシャドウ | ゲームUIのチープさにつながる |
| Font Awesomeなどのアイコンフォント | 世界観と合わない汎用感が出る |
| パララックス背景移動 | 素材ゼロのため実装不可。速度感も世界観を壊す |
| ハンバーガーメニュー | LPのため不要。ナビゲーションは最小化 |
| カウントダウンタイマー | 商業感・煽りが出る |
| ソーシャルプルーフ（レビュー数・フォロワー数） | 数値化による没入感の破壊 |
| 過剰なスクロールアニメーション | スクロールのたびに要素がポップインする演出は禁止 |
| 純白（#FFFFFF）・純黒（#000000） | コントラストが鋭すぎて没入を壊す |

## 参照方向性

**世界観・トーン**
- IMMORTALITY（Sam Barlow）——断片的な情報提示、解釈の余白
- What Remains of Edith Finch——静かな語り、喪失の美しさ
- BEFORE YOUR EYES——感情的共鳴の設計、説明しない強度

**ビジュアル参考パターン**
- `web/templates/pattern-H_dark-immersive.html` — ダーク没入型の基本設計

- 文学的・映画的な質感
- 説明を排し、問いと余白で語る
