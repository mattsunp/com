# ブランドルール — Seep Logos

## トーン & ボイス
- **基本姿勢**: クリエイティブで情熱的。世界観への深いリスペクトを持つ。
- **文体**: 丁寧かつ専門的。過度にカジュアルにならない。社外向けは「です・ます」調。
- **キーワード**: 「創生」「世界観」「没入感」「クリエイター」「エンタメ」「IPコンテンツ」
- **数字の書き方**: アラビア数字（1, 2, 3）を基本とし、文中の単位は半角スペース不要

## 禁止表現・禁止行為
- 競合他社の具体的な名指し批判は禁止
- 未発表IP・タイトルの情報を外部向け文書に記載禁止
- 「普通」「一般的」「よくある」等、世界観の独自性を損なう表現を避ける
- 過度な謙遜表現（「弊社程度では…」等）は禁止
- 絵文字・顔文字の使用は SNS 以外では原則禁止
- 「革命的」「業界初」等の根拠なき誇大表現は禁止

## ブランドキーワード（積極的に使用）
- **「没入感」** ← 全作品・全部門の最上位キーワード。常に意識すること
- 「世界を創る」「物語を紡ぐ」「クリエイターファースト」
- 「世界観ドリブン」「IP創生」「没入型エンタメ」

## SNS・マーケティングにおける「現実」と「架空世界」の分離原則

Seep Logos の作品プロモーションにおける最重要ルール。

**1投稿内に両者が共存すること自体は許容する。ただし、文章の流れの中で境界を明確にセパレートすること。架空の世界の言葉と現実の告知文が溶け合った状態で並ぶことを避ける。**

| 投稿要素 | 役割 | 例 |
|---|---|---|
| **世界観パート**（架空世界側） | 世界・キャラクター・物語の断片を届ける | 詩的な一文、世界の空気感、キャラクターの言葉 |
| **告知パート**（現実側） | 情報解禁・サイト誘導・リリース告知等 | 「公式アカウントでお知らせします」「詳細は公式サイトへ」 |

**セパレートの方法：**

区切り線（`──────` 等）は使わない。空行による余白で境界を作る。
線は主張しすぎる。空白は、ただそこにある。

```
傘は、雨粒から守るためだけにあるわけじゃない。
                        ← 世界観パートはここで完結。次の行には何もない。

新作ゲーム『Umbrella』近日公開。
続報は公式アカウントでお知らせします。
                        ← 告知パートはここから
```

**なぜ区切り線ではなく空行か：**
Seep Logos の美学は「浸透（Seep）」にある。世界と現実の間に壁を立てるのではなく、
ふっと息を吸うような間を置くことで、受け手は自然に切り替わる。
境界は感じられるが、主張しない。これがSeep Logosの文体における空白の役割である。

## 世界観・空想の扱い方
- **世界観構築人**は「現実から地続きの社会の未来視」を本質とした世界観提案を行う
- **空想要素の挿入**は暫定的にシナリオ書きが担う。空想はドラマの必然から生まれるべきであり、「ドラマに都合のよい装置」として使うことを禁じる
- 空想が「悪い意味でドラマに都合のいいもの」になっていると感じた際は、プロデューサーまたは各エージェントが随時指摘する
- 将来的に空想要素の専門エージェント新設を検討する（作品が具体化し工程がボトルネックになった段階で判断）
